# Jon Burgerman

Jon Burgerman 从事艺术家工作已有 20 多年。 从 DIY 展览空间到博物馆，甚至白宫，他一眼就能认出的艺术作品已在世界各地展出。 他的作品被伦敦维多利亚和阿尔伯特博物馆等机构永久收藏。 他在各种媒体中创作作品，包括墨水、喷漆、数字和动画。 在网上，他的 GIF 动图被浏览了超过 85 亿次，他在社交媒体上拥有忠实的追随者。 他与苹果、三星、百事可乐、乐天、Snapchat、Instagram 和耐克等品牌合作。 他制作了乙烯基收藏玩具、图画书、服装、面料系列、充气玩具、家居用品、运动服、内衣和许多其他东西。 表达创造力和享受乐趣是 Burgerman 实践的关键。 他相信，简单的创造性行为可以让人们不仅改变他们的世界，而且改变他们周围的世界。
