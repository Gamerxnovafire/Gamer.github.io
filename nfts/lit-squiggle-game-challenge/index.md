# LIT Project Three - Squiggle Game

LIT Squiggle Game 是对最初深受喜爱的手机游戏 Nokia Snake Game 的重新构想。

我们希望将 Snake 游戏带入未来，正如 Snake 游戏标志着移动游戏的新时代一样，Snowfro、他令人难以置信的团队和 Artblocks 的一群出色的艺术家开创了链上艺术和文化表达的新时代。通过这场比赛，我们庆祝他们俩。

就像诺基亚一样，LIT Squiggle Game 专为移动设备而设计。直接来自您的 Opensea 帐户。

请访问官方 LIT Discord 或 Twitter 帐户，了解有关如何玩和参与 LIT Squiggle 游戏挑战赛的更多信息
