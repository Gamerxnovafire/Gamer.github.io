# Meowniverse - Meowtizen

Meowtizen 是生活在以太坊区块链上的 3333 只随机生成的 Meowiverse 不寻常的猫。

Meowtizen 是第二代 Meowniverse 系列，拥有超过 450k 种可能的组合。

Meowniverse 是一个社区驱动的 NFT 项目，分享喵的爱。我们真的想给我们的持有者带来好处，让他们有动力“持有”，而不仅仅是纸上谈兵。

Meowtizen 是 Meowniverse 中的第二个系列。以太坊区块链上将随机生成 3333 只不寻常的猫。

Meowniverse 品牌：我们期待通过专注于奠定全渠道基础，将 Meowniverse 与现实连接起来。根据趋势以及现实世界与 Web 3.0 之间的联系，大脑的形式会有所不同。
