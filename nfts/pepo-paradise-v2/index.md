# Pepo Paradis

PEPO 的 Cheeks 是 3,888 个可爱的独特 PFP 的集合。Cheeks 是我们在 PEPO 天堂孵化的第一个 NFT 品牌，并将成为第一个成为 PEPO 之一并启动 PEPO 生态系统的系列。PEPO Paradise 是一个围绕你已经拥有的 NFT 的跨链社交 Petaverse，具有新颖的 Feed2Earn 生态系统。由屡获殊荣的游戏工作室基于 AVAX 构建。

宠物口袋乐园（PEPO Paradise）是一款跨链 Petaverse，在 Avalanche 上具有独特的“FEED-TO-EARN”机制。此外，PEPO 天堂将web3 虚拟宠物(PEPO) 联合到不同的链和元节中，并为交互式、可互操作的 web3 未来开辟了新机遇。
