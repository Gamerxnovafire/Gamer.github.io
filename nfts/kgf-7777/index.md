# Killer GF 7777

Killer GF 是由前 Riot Games 艺术家 Zeronis 创作的 7,777 个生成肖像艺术系列，拥有超过 240 个精心设计的特征，这些特征与可爱且具有美学吸引力的女友形成鲜明对比，这些女友也恰好是危险的刺客。

什么是杀手GF？
Killer GF是一个NFT（不可替代令牌）集合。存储在区块链上的数字艺术品的集合。
