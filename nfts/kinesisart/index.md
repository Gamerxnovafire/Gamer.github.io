# Kinesis Art

Kinesis Genesis系列《Planar Explorations》目前正在进行中，由两部主要作品组成：《Atlanta》和《Kafka》——Genesis系列总共将有5部作品随着 NFT 技术的出现，数字艺术的兴起打开了通往机遇世界的大门——数字媒体使我们能够超越艺术及其静态之美，并将艺术作为一个活生生的呼吸系统来体验。Kinesis 是一家实验艺术工作室，通过发行一系列有限的、精心策划的公共铸币厂，探索通过数字艺术形式产生的运动。我们有两个目标：

1. 开发一个强调艺术、社区和所有权的平台
   2. 专注于我们认为独特美丽的数字艺术方面：运动

在追求这些目标的过程中，我们希望引领数字活力的新时代
