# Snook

玩家在游戏中的斯诺克角色是铸造的 NFT。随着玩家在游戏中的进步（尽可能吃掉所有东西，避免和杀死其他斯诺克人并获取他们的特性），他会累积成就（例如，特性和稀有/难以获得的特殊皮肤）。区块链和加密技术使我们能够一成不变地记录这些成就（将其视为区块链/加密“桶”）。游戏角色/NFT 可以在未来的游戏中访问或用于商业目的，并且 - 在某些条件下 - 为玩家提供经济激励。
