# The Weeknd x Strangeloop Studios

为什么我们喜欢The Weeknd我们非常高兴地欢迎 The Weeknd 来到 Nifty Gateway，向全世界发布他们的创世纪 NFT 系列。作为地球上最具创意和突破界限的艺术家和音乐家之一，我们非常高兴 The Weeknd 加入 NFT 空间。Acephalous 是 Abel “The Weeknd” Tesfaye 和洛杉矶设计团队 Strangeloop Studios 的 8 件作品的集合。自 2015 年以来，Weeknd 和 Strangeloop Studios 就在音乐会视觉和视听作品上进行了合作，建立了一条在每个专辑周期和巡回设计的风格排列中始终保持一致的美学主线。

通过对新未来主义和反乌托邦叙事之间对话的共同热爱，以及看到数字空间中呈现的自然形式，The Weeknd 和 Strangeloop Studios 在流行艺术中开发了一种神话和文化的共同语言。Acephalous 中的作品探索了这些关键情感：The Weeknd 的“源”造型，呈现在一个空旷的空间中，并在时间流逝留下的各种“人工制品”和“残余物”中发现。艺术的价值不仅保存在“实际的”材料或图像中，还保存在图像所表示的意义和记忆中。

The Weeknd 和 Strangeloop Studios 之间的创造性对话一直是音频和视觉之间交叉影响的通感过程。NFT 的媒介为团队提供了在具有前所未有的自主权的新空间中进行协作的机会，在这里，创作过程的双方都可以蓬勃发展并相互增强。Acephalous 的“无头”含义既指区块链社区的去中心化治理，也指作品中被斩首的面貌，与其初始价值的来源相分离。

该系列共有 8 件作品：一次 1/1 拍卖，包括 The Weeknd 未发行的歌曲和“来源”造型；三部视听作品作为开放版提供，每部都包含同一首歌曲的一段经过过滤的效果，以及三件“人工制品”的视觉效果；以及四件静态艺术品，以 2x 无声拍卖和 2x 图纸的形式提供。

每件作品的描述都包含来自 Ta-Nehisi Coates、Judith Butler、George Battailles、Susan Sontag、James Baldwin 和 Walter Benjamin 等思想家的题词，这些题词激发了视听作品的主题张力：原创与复制、稀缺性与可访问性，记忆与神话。
