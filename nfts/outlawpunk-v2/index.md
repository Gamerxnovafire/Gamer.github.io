# Outlawpunk V2

我们与其他 NFT 项目最大的不同在于持续发展，Outlawpunk 是真正的社区项目。传统的 NFT 项目将在第一批铸造后停止开发，但对于 Outlawpunk，我们将继续完成我们的背景故事，并为铸造的不同 Outlawpunk 添加犯罪描述。


那些拥有传奇和稀有的 Outlawpunks 的幸运成员仍然会赚很多钱。然而，随着更多的发展，那些有共同的歹徒朋克的人也可能有更高的底价，因为更多有趣的罪行与不同的歹徒相关联！

这是一个基于社区的项目，更多有趣的犯罪创意将从社区中收集，因此这种发展将永无止境；从长远来看，这可以推高楼面价格并创建一个更强大的社区。
