# THP Zine

Heart Project Community Zine 是来自 Heart Project 才华横溢的成员的多媒体艺术作品集。 在我们的第一期中，我们通过一系列色彩缤纷的照片、图画和图形设计来探索爱情的复杂本质。 感谢所有提交此问题的才华横溢的爱心人士！ 还会有更多。 Aidan Cullen 和 Henock Sileshi 的创意指导，Alan Moutal 的动画，D33J 的配乐。

The Heart Project 是一个社区经营的创意工作室，以一种新的方式促进艺术合作。 以太坊区块链上的 10,000 个独特代币可用作会员通行证，让持有者能够获得赠款和创造性贡献的机会，并能够共享这些团体创作的所有权。
