# Wagmi World OfficiaI

Wagmi World 由一支多元化且富有创造力的团队经营，他们在多个行业和专业拥有多年经验。话虽如此，没有我们出色的监护人和社区助手的帮助，Wagmi World 是不可能的。随着加密货币和 NFT 世界开始爆炸式增长，我们的使命是成为新人安全进入这个快速发展的空间并通过教育和机会赋予网络权力的桥梁。

我们将通过制作有关如何安全浏览 WEB3 以及如何在 NFT 领域明智地投资时间和金钱的教育内容来实现这一目标。

我们意识到前 2 个月对于 NFT 项目至关重要，这就是我们选择规划 2 个不同路线图的原因！我们不想在不知道我们的社区首先想要什么的情况下过度承诺我们的未来，这就是为什么社区投票将有助于指导我们的原因。
