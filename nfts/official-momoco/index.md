# Official Momoco

羽生从小就和莫莫科一起长大。他们从白天到黑夜，从春天到冬天在浅草寺共度时光。然而，浅草寺百年不变的风景，却让羽生有了走出去的念头。然后，他离开了五年，没有任何回应。

Momoco从小就有成为世界顶级画家的梦想。和她一起长大的羽生结弦，不仅是她梦想背后最大的支持，更是她默默爱着的人。她认为，这辈子和她爱的人一起守住浅草寺的周围就足够了。然而，羽生离开浅草寺五年。 Momoco等了五年。

当浅草寺的风铃在桃子头顶叮当作响时，她坐在浅草寺门口的台阶上，抬头看着风铃发呆。清澈的阳光透过钟摆缝隙照在桃子脸上，让她恍惚间，她发现羽生离开已经五年了。五年来，她一直在这里守候，希望有朝一日能见到那个熟悉的人。她听说浅草寺的近签可以帮助人们美梦成真，每个月桃子都会向大神请教近签，但她最想要的似乎是故意躲着她，怎么也找不着。她非常寻找它。

经过多年的等待，Momoco 终于放弃并改变了主意。她希望自己和羽生的记忆可以换一种方式永远存在，于是桃子在画纸上写道：“人生有七难：生、老、病、死、爱、分离、和你的人一起生活。”怨恨，得不到你想要的。”但在莫莫科的作品中，“爱”是最难的。有了爱，世间万物便有了。 Momoco NFT是浅草寺一万条过去记忆的集合，带来东京浅草寺的加持。在这部作品中，祝愿所有的有情人都能健康快乐的生活。
