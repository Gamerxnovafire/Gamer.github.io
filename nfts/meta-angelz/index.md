# META ANGELZ

欢迎。 META ANGELZ 是随机生成的生活在 META HEAVEN 中的独特角色。 META ANGELZ 是一个 NFT 社区，面向想要学习如何生成 NFT 并获取 NFT 信息的人们。 我们的目标是聚集所有 NFT 创作者，并有机会加入我们的社区和 NFT 比赛，称为“META ANGELZ LEAGUE”。META ANGELZ LEAGUE 的获胜者将获得营销服务、铸造、获得冠军徽章 NFT、免费展示等福利 在名为 META HEAVEN GALLERY 的 metaverse 中。对于那些对 NFT 有很好的想法的人，META ANGELZ 将帮助您在 NFT 上取得成功。
