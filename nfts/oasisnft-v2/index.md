# The OasisNFT

关于绿洲
自由、冒险、创造

绿洲是由一群崇尚自由创意、反叛传统的探索者组成的。下一代互联网革命将我们带到这里。我们将从 NFT 出发，建立一个平等自由的社区，从这里开始挑战未知领域的冒险。

愿景与价值观

我们的愿景是成为虚拟世界中最重要的游戏社区。
价值观：
→ 社区至上：社区是我们前进的最重要力量，我们始终将社区价值放在首位。
→ 相信时间：好事需要时间。前面的路是曲折的。也许我们会遇到困难。有社区的支持，我们将无所不能。
→ 敢于探索：我们不会被过去束缚。我们不会因为恐惧而停下脚步，探索者的基因铭刻在我们的血液中。
