# RAGS to RICHIE by AIec MonopoIy

从很小的时候起，亚历克就被艺术和创造力所包围。亚历克在纽约这样一个充满活力的城市长大，很快就开始四处探索，并在附近结识了许多街头艺术家。他独特的风格使他的壁画具有辨识度，并迅速在纽约多个地区声名鹊起！

亚历克周游世界，在洛杉矶、迈阿密、香港或纽约等许多主要城市绘制他的壁画。他是一位国际知名的艺术家，他的作品受到许多艺术和 NFT 专家的好评！

Alec 很快意识到今天在 NFT 领域创造的东西的重要性。他被 NFT 专家和 Web3 创作者很好地包围着，现在自己也是一名狂热的 NFT 收藏家。对于 Alec Monopoly 来说，未来在于 Web3。
