# Lure Fish Club

第一款基于EVM Chain的Web3.0钓鱼游戏。
在 Beta 轮中自由发挥
诱鱼俱乐部
又名 L.F.C 或 L.F 俱乐部

  钓鱼竿可用于钓鱼、制作和升级。
  宝箱和钓鱼竿是 NFT，可在 NFT 市场上交易。
  钻石是治理代币，黄金是游戏代币，都可以在二级市场交易。
