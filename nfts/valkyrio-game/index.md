# Valkyrio Game

受漫画启发的基于区块链的游戏。
在 Valkyrio，玩家将通过招募战士、锻造武器、杀死怪物和挑战众神，沉浸在人类代表的角色中。从那时起，每个参与者都将贡献自己的故事情节。
在 Valkyrio，我们相信拥有独特的战士和/或武器是游戏具有吸引力和吸引力的方式之一，同时也是对玩家的奖励。有一定程度的随机性，也有一种主人翁感。VALKYRIE 现在是可选的
