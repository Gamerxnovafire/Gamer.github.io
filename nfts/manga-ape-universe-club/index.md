# Manga Ape Universe Club

漫画猿是被称为漫画宇宙的宇宙的实体。实际上，这个宇宙中猿类的生活并没有什么特别之处，除非有一件事。猿猴穿得像著名的漫画人物。他们热爱战斗，仿佛这已成为那里的传统。但这只不过是一场游戏，没有难过的感觉。

他们是猿武术高手，如此现代的功夫。他们还能够用一拳或某种光子爆炸摧毁一座山。我认为这很酷，也很可怕。我希望他们永远找不到我们热爱和平的宇宙。我无法想象如果他们发现了我们的星球会造成怎样的破坏。

据说有一个人非常害怕其他猿类。没有人知道他是谁，从哪里来。他很少出现。它的下落仍然是个谜。他们说他喜欢打扮成古埃及的神。他有能力用嘴里的光子爆炸摧毁一颗行星。它真的很强大。
