# Shibance

Shibance 生态系统
Shibance 旨在建立一个专门用于 meme 令牌的整个生态系统。我们的原生代币是 Shibance 代币，代码：WOOF。
启动功能：

  DEX平台
  合作项目的农业奖励
  WOOF 质押池

发布后：

  初始 WOOF 产品（“IWO”），在我们的启动板上
  WOOF 彩票
  每周对 meme 代币进行农业提升的投票
  绝密开发更新（项目代号：FLOOF）
  NFT 农业
  跨链支持

我们的团队由前端和后端开发人员、DeFi 专家、社区经理和专门的图形设计师组成。项目将经过仔细审查，以确保我们只列出最好的模因，并将地毯的风险降至最低。
