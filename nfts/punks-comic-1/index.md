# PUNKS Comic

认识朋克！每个人最喜欢的不合群的收藏家、反叛者和来自元宇宙的加密货币 degens。Pixel Vault 的一个项目。

在 PUNKS 3 事件之后，沮丧的特工 Ivanova 回到办公室，与此同时，PUNKS 工作人员制定了一个计划，利用与 FBI 特工有着共同过去的希拉里来帮助她摆脱困境。但是这个计划会奏效还是会进一步激怒这位火爆的能手特工？

出于将 NFT 带入生活的愿望，PUNKS Comic 为元界中一些最受欢迎的 NFT 系列提供了名称、身体、背景故事和丰富多彩的叙述。介绍 Courtney、Marvin、Frank、Elvira 和 PUNKS 的主要角色。
