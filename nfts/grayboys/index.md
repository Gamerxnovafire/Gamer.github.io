# Gray Boys

Gray Boys 是 10,000 个生成 NFT 的集合，它们是 The Mothership 的成员。

Gray Boys' Mothership 是第一个将 NFT 所有权验证、其他蓝筹项目的特权和效用代理给所有 Grey Boy 持有者的实验。Grey Boy 持有者投票支持 The Mothership 的钱包获得的 NFT，并且所有成员共享这些获得的 NFT 的代理所有权。
