# Warped Ape Yacht Club Official

Warped Ape Yacht Club 是 9999 个 NFT 的集合。随着整个薄荷糖的售罄，WAYC 正在通过 teamtrees.org 种植 20,000 棵树，并通过teamseas.org 清理 20,000 磅的海洋污染。WAYC 也在制作一个简单但令人上瘾的 REFI 游戏。

REFI 代表再生金融，它使用融资作为解决系统性问题的工具，用自然环境再生社区，恢复自然的平衡。

我们要去元界！我们现在正在谈判并与其他项目合作，在元宇宙中拥有我们自己的真正的游艇俱乐部和 NFT 博物馆。我们想要 VR 体验，我们想要在另一个层面上真正与我们的社区建立联系！
