# Paradise Ticket - Pepo Paradise

宠物口袋乐园（PEPO Paradise）是一款跨链 Petaverse，在 Avalanche 上具有独特的“FEED-TO-EARN”机制。此外，PEPO 天堂将web3 虚拟宠物(PEPO) 联合到不同的链和元节中，并为交互式、可互操作的 web3 未来开辟了新机遇。

Paradise Ticket 是一个包含 1000 个 NFT 的集合，旨在奖励我们在 PEPO Paradise 社区中最早的支持者。门票为持有者提供各种特权，以确保他们获得顺畅的体验并提前进入 PEPO 天堂游戏。

PEPO Paradise 是一个围绕你已经拥有的 NFT 的跨链社交 Petaverse，具有新颖的 Feed2Earn 生态系统。由屡获殊荣的游戏工作室基于 AVAX 构建。
