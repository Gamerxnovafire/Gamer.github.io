# Uniswap V1

Uniswap 是一种在以太坊上进行自动令牌交换的协议。 Uniswap 协议在设计时考虑到了简单性，为在以太坊上无缝交换 ERC20 代币提供了一个接口。通过消除不必要的租金提取和中间商形式，它可以实现更快、更有效的交换。在它进行权衡的地方，权力下放、审查阻力和安全性是优先考虑的。开发者、交易者以及流动性提供者共同参与一个对所有人开放且可访问的金融市场。
