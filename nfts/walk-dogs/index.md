# Walk Dogs

Walk Dogs 是一款内置 Game-Fi 元素的 Move-To-Earn Web3 应用程序，它结合了我们生活中很重要的两项活动：步行和赚钱。 Walk Dogs 协议允许用户在使用真实和虚拟 NFT 宠物的每一步中赚取 $WLD 代币。为了让 Move-To-Earn 的过程更加有趣，该应用程序将包含各种游戏。在单人模式下，用户将使用他们的 NFT Dog 来行走和赚取代币。用户步行的速度会影响他们的总行驶距离。在双人模式下，用户将使用他们的 NFT 狗并与同样拥有 NFT 狗的朋友一起散步，这样双方都可以赚取更多的代币。
Walk Dogs 生态系统使用 $WLD 代币，它是主要的实用代币。它用于移动应用程序和 DApp 中，用户将在其中铸造他们的 NFT 狗，在市场上购买和出售 NFT，最重要的是走路并赚取 $WLD 代币。 Walk Dogs 生态系统包括一个 DApp、一个移动应用程序和一个 NFT 市场。
该项目的目标是在数百万人中推广健康的生活方式，并激励他们与毛茸茸的朋友共度更多时间！该团队很自豪地向用户展示了当今人们可以通过 Web3 技术实现的潜力，让用户有机会过上积极的生活方式并在此过程中获得丰厚的奖金！
