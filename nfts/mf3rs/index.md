# mf3rs

mf3rs 是 McGill 对 CC0 的一次硬核 3D 庆祝活动。

CROAKZ 的灵感来自 CrypToadz，并将在元宇宙和肉类空间中蓬勃发展。

质量第一。CROAKZ 将拥有精雕细琢的卓越感觉。达到这个标准的范围超出了 CROAKZ 薄荷糖。当我们接近元宇宙时，机会是无穷无尽的。所以让我们准备好跳跃吧！一旦我们的团队认为 CROAKZ 系列达到并超出了我们的标准和期望，我们将立即公布薄荷的详细信息。

CROAKZ 非常适合在 Meatspace（物理、有形世界）中采用。我们将寻找专门生产优质乙烯基模型和玩具的知名工作室，梦想有形的 CROAKZ 很快就会在您的办公桌上冷却下来。我们处于早期概念阶段。对有形 CROAKZ 的部分探索将包括可 3d 打印的 CROAKZ、软糖、糖果、帽子、衬衫、补丁等等！
