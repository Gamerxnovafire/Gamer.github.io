# ursaparty

什么是熊党？
ursaparty 是一项仅限受邀参加的活动，所有熊类最杰出的代表每个世纪都会聚集一次，以拯救他们的物种并度过愉快的时光。

repondez, s'il vous plaît
ursaparty 将在以太坊区块链上举行，6464名随机生成的 ERC-721 熊将出席。

每个ursa都是由100 多个不同稀有性的特征生成的。
此外，6位特邀嘉宾1/1熊也将出席。
