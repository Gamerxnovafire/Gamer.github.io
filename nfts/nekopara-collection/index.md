# NEKOPARA Collection

Chocola (ショコラ Shokora?) 是一个非常开朗直率的猫女，性格像狗，非常喜欢她的主人。 她是Vanilla的双胞胎妹妹，是Minaduki猫女中最小的一个。（如9个月大）她不太擅长处理复杂的事情，但她总是友好而充满活力。 她倾向于在完全考虑清楚之前就采取行动。 尽管她有一些笨拙的时刻，但她积极进取的态度赢得了她周围每个人的极大爱戴。

Chocola 一头黑发长发，扎成两条双马尾，头上系着两条小丝带，与她的红白连衣裙相配，搭配长筒袜和粉色鞋子，比她穿的其他颜色都要亮。

她收到了一个银色的铃铛，尽管没有提到猫女在参加铃铛考试后获得的颜色对应的等级或地位。
