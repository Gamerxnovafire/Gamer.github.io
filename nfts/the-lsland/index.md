# THE lSLAND

新系列“THE ISLAND”是一个现实，似乎不是一个系列的五部作品，而那个似乎在那里的形象几乎像戏剧的第四幕一样面对着最终定理，但仍然不清楚 ，象征寂寞抛入大海，或心中渴望。 这种欣赏是将感受的一部分传递给观众，而不是作者解释它，所以无论你对作品的感觉如何，都可以完成作品。

回顾复杂的内心世界，交织在生命与时间的关联中，例如无法摆脱过去、现在和未来时间的命运。

我的观点一个系列，通过山的形象表达了与无法逃脱的人类的巨大自然的交流，以及通过自然感受到和运作的生命、痛苦和解放。

抽象与思想是否不能共存的问题是每日梦想系列的开始。 在东方，强调阴阳的和谐，宇宙万物和人类的一切现象，都是用这个阴阳来解释的。 可以说，既接受西方的理性与感性的一面，又接受东方阴阳的适当平衡，符合中庸之道。 我的绘画旨在将抽象和思想融合在一个地方，并在一个屏幕上实现完美的和谐，而不会与另一个屏幕相互补充和伤害。
