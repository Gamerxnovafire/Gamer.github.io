# PolyDoge X Polycat

PolyDoge 社区非常喜欢它。这也意味着 PolyDoge 的大部分循环供应被锁定在农场中。这对所有持有者都有好处。
考虑到这种合作关系，我们决定通过限量版 NFT 来纪念 PolyDoge 和 Polycat 之间的特殊关系，以纪念那些将 PolyDoge 投注在 Polycat 的人。
我们还包括那些将他们的 Polycat 放在那里的 Polycat 社区成员（向他们介绍 PolyDoge！）——最后一组是 PolyDoge LP（因为他们是我们社区的真正英雄。）这是一个特别酷的 NFT 下降。这是一款令人眼花缭乱的 3D 戒指，带有 Polycat 和 PolyDoge 徽标。它也是动画的，看起来绝对令人惊叹。感谢我们的常驻 PolyDoge 艺术家 Colin Perras 让这一切成为可能。
哦，每个戒指都有自己独特的额定功率- 越高越好。
