# Pudgy Penguins 3D Official

Pudgy Penguins 3D 是一个生成集合，包含 1000 个独特的企鹅 NFT。该系列将分 4 个阶段发布，每

个阶段仅发布 250 只矮胖企鹅。每只矮胖企鹅都是独一无二的，永远不会有两只完全相同的矮胖企

鹅。拥有一只矮胖企鹅 3D NFT 赋予所有者转售其 NFT 的独家能力，并为该系列的所有未来转售设定

自己的版税金额。已经创建了一个 OpenSea 页面，供所有拥有 Pudgy Penguin 的 NFT GO 成员访

问，以便列出他们的 NFT 进行转售。对于那些选择持有他们的 Pudgy Penguin 的所有者，他们将成

为全球所有者的独家社区的一部分，随着 NFT 元界连接的扩展，他们可以将他们的 Pudgy Penguin 

NFT 导入元界。
