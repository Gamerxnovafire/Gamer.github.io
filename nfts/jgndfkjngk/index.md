# MINDBLOWON UNIVERSE

元节中 6969 名怪人的另一个 Mindblowon 日，来自 Mindblowon 工作室的 Lickpalik 的 Tahilalats 的创建者。加入我们，踏上一段奇特但幸福的旅程，前往一个充满想象力和欢笑的新宇宙。让我们开始吧！

LickPalik 只是一个有着不同寻常的观察力、喜欢画画的孩子。2015年，他开始根据日常生活画漫画。他称这部连环漫画为 TAHILALATS；印尼语为皮肤痣；作为基于他自己脸上的皮肤痣的身份标志。那年晚些时候，韩国数字漫画出版商 LINE Webtoon 与 LickPalik 接洽，希望他成为一名投稿人。在四年的时间里，TAHILALATS 发布了超过 653 章，获得了 3730 万个赞，成为平台上的顶级喜剧内容。

现在，随着超过 3000 部在线漫画的出版，TAHILALATS 不仅仅是一个连环漫画，它成为一个杰出的喜剧品牌和拥有数百万粉丝的 IP，并将其业务扩展到创意工作室服务、内容开发、品牌授权和合作，商品、棋盘游戏、玩具，现在潜入成为 Mindblowon Studio 旗下的 WEB3 品牌。

TAHILALATS 相信，世界各地的人们在日常生活的喜剧中都有一线希望，我们的目标是逗逗其中有趣的骨头。我们才刚刚开始，我们将探索新的世界和维度，例如从 Collectible PFP NFT 开始的 Metaverse 等等！
