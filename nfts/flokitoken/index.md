# FlokiTOken

从前，有一只非常特别的狗。那条狗是一只 Floki，这条狗启发了全世界数以百万计的人将资金投入到带有狗形象的代币中。
根据#FlokiArmy 的说法，FLOKI 是“狗狗币杀手”，并将在他们自己的 NFT 市场和彩票系统中列出。
FlokiToken 是专注于 NFT 市场和彩票系统的狗爱好者的 meme 令牌。
