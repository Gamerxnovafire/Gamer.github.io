# Wallkanda Polygon

街头艺术是 21 世纪的主要艺术运动，但街头艺术家正在努力以他们的艺术为生。他们没有像 Instagram 或 Pinterest 那样被困在 web2 维世界中的其他数字工具。
我们通过为艺术家提供新的收入来源来帮助他们拥抱下一代互联网。我们试图通过三个主要领域来实现这一目标：

  捐款就像图片一样简单
  他们艺术的不可替代标记化
  与 DeFI 更好地集成

为了实现这一目的，我们正在创建一个可验证的去中心化艺术家注册中心，以促进他们在 web3 世界中的工作得到认可。
该注册表允许免费和轻松地铸造他们的作品。
艺术家现在将重新获得对他们作品的数字表示的所有权，通过他们的 IP 获得版税，并将他们的艺术作为金融产品的抵押品。
