# Rebel Boodle

Rebel Boodle 是 6190 个坏蛋 Boodle 的 NFT 集合，生活在以太坊区块链上。这是一个社区驱动的项目，旨在在 web3 和物理世界中建立一个可持续的品牌。拥有这款 NFT Grand 的您可以访问独家的 Discord 社区，它也将为 NFT 持有者带来欢乐。

拥有 RB NFT 将使您获得 Boodleland 赠品和竞赛的独家访问权。持有者将有机会直接赢取手绘 1/1 NFT、手机和以太坊到您的钱包。我们将使 RB 栩栩如生，并使其成为现实。持有者可以期待来自世界各地的聚会和展览场所。它将让持有者彼此互动并推广品牌。RB 计划为 NFT 持有者创造一种完全沉浸式的体验。我们将在沙盒中投资和购买土地。这将包括一系列角色、配件、可穿戴设备等等。
