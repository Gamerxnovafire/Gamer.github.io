# Matryoshka Hub Adventure Club

Matryoshka Hub 冒险俱乐部是一个创世 NFT 集合，包含 8888 个独特的随机生成的 2D 像素艺术 Matryoshkas，它们生活在以太坊区块链上。 就像区块链一样，每个套娃都有自己的散列，它是基于令牌 ID 和父套娃散列创建的，除了第一个使用 unix 时间戳而不是父散列的散列。 最好的俄罗斯套娃图像将用于街头艺术项目。您从 MHAC 收藏的 NFT 是您加入我们 Discord 冒险俱乐部的会员卡。访问 www.MatryoshkaHub.com 了解更多详情。
