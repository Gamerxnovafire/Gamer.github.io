# Defi Empire games

Defi Empire 是币安智能链上的一个生态系统，每个人都可以在一个安全、值得信赖的环境中享受最流行和最有利可图的 Defi 项目/游戏，并提供透明的无地毯合约。
我们将逐步发展生态系统，每 2 个月至少发布 1 个新功能，开发用户可以投资的去中心化应用程序，从我们的矿工游戏开始：有抱负的冒险家 Jayce。
