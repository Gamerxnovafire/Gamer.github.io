# Marso Genesis Block

Marso 展示了另一组作品，结合了优雅的几何形状和充满活力的调色板，被称为区块链的“创世纪块”，让人想起著名的西斯廷教堂的天花板。 这个系列既可以作为一个单独的作品独立存在，也可以作为一个整体独立存在，每件作品都是独一无二的，从而导致“创造”的高潮和整个区块链空间的建立。 话虽如此，整个系列可以从左下角（创造）、中下角（分布）、右下角（进步）、左上角（繁荣）、中上角（自由与自由）、右上角（ 社区），中右（沉浸），中右（整合），并在中间结束（伊甸园）。
