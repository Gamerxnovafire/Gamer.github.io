# Square Cats Club

Square Cats Club 是以太坊区块链上 9999 个 Square Cats 的通缩 NFT 集合。 拥有者可以将 3 只方形猫组合成方形猫战士！ 所有者将能够在独家角色扮演体验中决定 Square Cats 的命运，在经过所有者验证的 Discord 频道中与志同道合的所有者交流，获得独家 Square Cat 商品等等！

直到最近，Square Cats 在以太坊区块链上的 Square Earth 上和平地生活和嬉戏，住在他们的 Square 房子里，和他们的 Square 小猫玩耍。然而，一个严重的危险已经威胁到他们的存在……圆狗。独自一人，方猫无法与这个敌人作战。他们必须联手成为方形猫战士！

为了成为方形猫战士，三只方形猫需要联合起来合并。Square Cat Warrior 装备更好，并且具有更稀有的特性来处理圆狗的邪恶力量。
