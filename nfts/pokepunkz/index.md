# PokePunkz

欢迎来到 PokePunkz 俱乐部。一个供收藏家一起享受社区交易和乐趣的地方。每个 PokePunk 都将在 ERC721 中铸造出来，使它们在我们迈向虚拟世界时可用于元宇宙开发进程。

PokePunkz 俱乐部由1151个 PokePunk Coin 游戏就绪 NFT 组成。

每个 PokePunk 都是在 ERC721 上铸造的，您可以放心，拥有这些 NFTS 会很有趣。每顶帽子都是在 24x24 像素模板上手工设计的，然后放大到 480x480 像素。每组 PokePunkz 均由随机背景、朋克朋克类型和 56 种不同 PokeHat 选项中的一种生成。
