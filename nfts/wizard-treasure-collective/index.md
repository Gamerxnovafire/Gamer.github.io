# Wizard Treasure Collective

什么是巫师宝藏集体？
WTC 是一个 DAO，它从最新的独家发行中购买和细分 NFT。 想想 Kaiju Kings、Doodles、Mutant Cats 等。然后这些分数以球体的形式分发给成员。

Orb 代币代表每个保管库中持有的 NFT 的一部分。 每个 Wizard NFT 每天通过质押赚取 10 个 Orb。

有多个避难所，每个避难所代表不同的策略，每个避难所都由不同颜色的球体表示。 每个 Wizard NFT 一次只能在一个 Vault 中质押。

Wizard Treasure Collective 将为持有者提供独家 NFT 掉落、定期可领取的 NFT 以及更多未来实用程序。
