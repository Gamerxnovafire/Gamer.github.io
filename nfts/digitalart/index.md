# DigitalArt

以太坊区块链上的世界上第一个区块链网站 NFT 收藏品市场现在可在 L2（第二层）Polygon 的区块链上使用。运营发展中的虚拟经济 DigitalArt.Crypto 代币“DANL”为我们的社区在 Web 3.0 和区块链网络中的 Metaverse 中增加 NFT 的价值。
