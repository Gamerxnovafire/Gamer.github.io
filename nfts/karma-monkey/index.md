# Karma Monkey

新的 NFT 集合提供 Monkeyverse 的成员资格、在 DAO 中投票资助 web3 和社区项目，以及访问任务和香蕉经济。Karma 艺术由我们获得奥斯卡奖提名的团队以及加密和技术资深人士共同创作。Genesis 是第一个在一次交易中创建的链上 pfp（头像）nft 集合。该系列具有简单干净的设计，同时实现了收藏家将永远欣赏的复杂而丰富的特征和元特征分布。Monkeyverse 是一群加密货币老手用 Web3 打造更美好未来的地方。OnChainMonkey (OCM) 社区包括主要区块链的创建者、世界上最大的加密货币交易所的创始人、元界的创建者、区块浏览器的创始人、保护相当大比例的比特币和以太坊网络的矿工等等。
