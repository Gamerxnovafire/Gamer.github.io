# Wolf Game - Land

有限的资源，只有 20,000 个创世地块将永远存在。每个都是独一无二的，具有不同的面积和可以与农民相乘的能力。在追求经济主导地位的过程中培养资产。欢迎来到狼游戏。我们正在共同构建一个由狼队统治的战略和博弈论世界。

在游戏中，你的成功和失败取决于你所做的选择。

迄今为止，在争夺珍贵羊毛的过程中，狼队和羊队面临着各种各样的挑战。通过最初的游戏，冒险游戏，阿尔法游戏，以及最近的洞穴游戏，玩家已经熟悉了游戏的方式，并且彼此熟悉。

我们一起做的。我们是先驱。我们是狼游戏。
