# Kea Farm

Kea Farm 是一款 NFT AgTech 游戏，任何人都可以同时成为玩家、投资者和农民。它诞生于将 NFT 世界与真实农业紧密联系起来的愿望，并作为一种向世界介绍农业和区块链技术的教育方式。我们相信赋予我们的玩家权力并为他们提供经济机会。
我们相信未来每个人都可以成为农民来创造自己的食物。最重要的是，我们有一个梦想，即我们的平台可以永远改变农业世界。
