# The China! NFT

第一套**5888**像素艺术头像。随后，**888**个 AI 城市和**88 个**独特的艺术呈现为三个不同的集合。

**怎么回事？**

社会评论和讽刺对朋克衍生品、人工智能艺术以及与政权或国家购买（或出售）相关的叙述。这件艺术品以女性、比特币、矿工、以太坊等数十种其他收藏品中没有的特征为特色。

**为什么？**

我们厌倦了刚刚停止尝试的隔夜衍生品。这些收藏品表明了这一切的荒谬性。如果“中国”更好呢？我们庆祝中国，它的人民，它的文化。无意冒犯。

**中国！**

*中国！*品牌将演变成另外两个**888**和**88**系列，所有权转让给持有人。
