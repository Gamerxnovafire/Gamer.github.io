# RENGA Black Box

这些盒子包含了解 RENGA 工厂的过去、现在和未来所需的数据。每个盒子都是通往未知世界的随机原型的门户。

使用此框打开门户并带出您的角色

每个黑匣子里都等待着一个新角色，一个新的收藏部分，以及一个要讲述的新故事。这些令人难以置信的独特手工艺术作品将与您最狂野的想象力一起使用，以构建 REGA 宇宙的奇迹。

由艺术家 DirtyRobot 开发的包含 10,000 个角色的手工制作集合。每个人都有自己的身份，可以在更广泛的 REGA 故事中被发现。最纯粹的 REGA 是艺术和讲故事的艺术。
