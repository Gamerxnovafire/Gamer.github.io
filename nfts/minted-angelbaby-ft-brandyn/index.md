# Minted (angelbaby ft. BrandYn)

“Minted”是休谟 NFT 生态系统中的第三款音乐 NFT。这 50 个限量版代币将让您在 Hume Genesis 的许可名单上占有一席之地，并且也是 Hume metastar 和世界上最著名的 Fluf angelbaby 的第三个音乐 NFT。这是第一部以艺术家 BrandYn 为主角的休谟音乐 NFT，其灵感来自 Quirkies 社区的精神。

以建立一个开放和创造性的元宇宙的名义，angelbaby 和休姆集体邀请了来自元宇宙的 Quirkies 出现在封面艺术上。Hume 集体成员和 Quirkies 持有者共同选择哪一个 Quirkie 将出现在 angelbaby 的封面艺术中。Quirkie #478 是在一个象征性的封闭式休谟集体决定中被选中的。
