# PLUR Official

Project PLUR 是一个拥有 8585 个代币的 3D NFT 项目。受嘻哈文化的启发，Peace Love Unity Respect，通常缩写为 PLUR。我们希望将这种精神传递给我们的社区，成为我们的精神力量。这个世界需要一点噪音，我们来了！
Web3 令人惊讶，但同时存在大量不良和欺诈性内容，我们想对此做点什么。我们的愿景是打造全球最酷的 Web3 品牌之一，其价值在于从每个 NFT 的形象中衍生出的高度持久的文化价值。每个 PLUR NFT 的版权归其持有人所有。因此，每个PLUR NFT持有者都可以进一步成为PLUR文化的创造者和传播者。我们希望 PLUR 意味着最高质量。很快，它将成为一种文化符号。归根结底，PLUR 是一种信念。
