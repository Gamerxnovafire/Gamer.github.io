# PUNKS: Origin Stories

在他们聚集在一起让不可替代的粉丝们惊叹之前，朋克们过着自己的生活，直到他们被召唤到 web3 中唯一的地方：起源城！ 跟随 Origin Stories，这是一部纯数字短漫画，探索 Courtney、GFunk 和 PUNKS 的明星（以及他们的同龄人）的背景，揭开他们是谁的层次，揭开让他们登上顶峰的阿尔法， 并了解是什么让他们走到了一起。 谁知道呢，也许我们甚至会知道 Wiggles 在他的过去有一两个秘密？

从最初的PUNKS漫画系列到我们最近推出的Origin Stories，解锁这个不合群的收藏家，叛逆者和加密degens背后的故事。
