# Pixel Howlerz

Pixel Howlerz 向 Noah Claire Davis 创建的原始 Howlerz 致敬。所有特征都是单独手工制作，然后由

代码生成以创建独特的图像。与 Howlerz 的艺术家没有直接关联，只是 Howlerz 的持有者和支持者。



##### ▶ 什么是 Pixel Howlerz？

Pixel Howlerz 是一个 NFT（非同质代币）集合。存储在区块链上的数字艺术品集合。



##### ▶ 存在多少个 Pixel Howlerz 代币？

总共有 618 个 Pixel Howlerz NFT。目前，194 位所有者的钱包中至少有一个 Pixel Howlerz NTF。
