# NoEvilMonkeys

加入并成为我们家庭的一员，让我们玩得开心！

该项目
该团队由两兄弟（AMA 和 Waalo）创立，他们有非裔德国血统，在德国柏林出生和长大。 他们与塞内加尔的紧密联系一直使他们意识到我们世界的巨大矛盾和不公正。

他们看到了使用 NFT 传播他们艺术的积极信息的机会，这是一种完美的交流方式。

他们与老朋友和庞大的网络一起，建立了一支拥有相同价值观并希望有所作为的团队。
