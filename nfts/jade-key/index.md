# JADE KEY

Jade Key 是一位以“Hardstyle”为主要流派的 EDM 艺术家和 DJ。

通过过去作为独立乐队“Sister's Barbershop”的原创成员的工作经验，JADE KEY 正在追求带有情感氛围的旋律的 EDM。JADE KEY通过全球唱片发行专辑，是一位志在登上世界舞台顶端的艺人。JADE KEY 从 2018 年至今发行了 +60 首歌曲，全球超过 250 万流（Spotify）。

2017 JYP作曲家试镜第三名

2018 年日本 Oricon 专辑榜第 18 位 - 'Vixx LR - Poison'

2019 以2019艺人名“JADE KEY”进入全球唱片公司

2020 世界 DJ 节 Untact 表演

2021 年希腊、土耳其硬核排行榜第 1 名 - “终于（英尺月神乐队）”

2022 与顶级厂牌 'Dirty Workz' 合作，Beatport 硬舞榜第 20 位“Can't Break Me (ft. KNVWN)”
