# Substrata

Substrata 是一个多用户网络空间/元宇宙。与其他用户聊天或探索其他用户创建的对象和地点。
您也可以创建一个免费的用户帐户并向世界添加对象！

Substrata 中的地块正在[出售](，可以铸造为以太坊 NFT。
