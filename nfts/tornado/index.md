# Tornado

什么是龙卷风？
TCORE 是一种基于 cVault (CORE) 源代码的非通胀加密货币，旨在以完全去中心化的方式自主执行盈利策略。

这些是 TCORE Vault 生态系统的主要产品：

通过 TCORE 质押实现非通胀农业
自动化策略库

TCORE 是一个通缩农业项目，由 cVault (CORE) 源代码分叉。然而，这不仅仅是一个简单的分叉——在原始 CORE 已经非常出色的基础上进行了许多更改。这些变化为 Vault 和 Pool 用户带来了最大的利润，并且该协议将不断开发和改进以造福投资者。
