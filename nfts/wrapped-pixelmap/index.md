# Wrapped PixelMap

最近重新发现的 NFT 项目于 2016 年 11 月 17 日在以太坊上启动。PixelMap 是第一个将图像数据直接存储在以太坊区块链上的 NFT。

每个图块都由购买它的实体真正拥有。因为数据本身，而不仅仅是一个指针，存储在区块链上，所以只有关闭每个以太坊节点才能消除数据。

合同的设计是为了在瓦片所有者想要更新图像、更改瓦片指向的 URL 或以他们想要的任何金额出售瓦片的情况下，他们可以在没有任何中央机构的协助或控制的情况下过程的任何部分。

如果 PixelMap.io 本身出现故障，每个像素的数据、所有者和 URL 都会保留在区块链上，并且可以轻松复制和显示整体图像，以及每个图块上的历史变化。
