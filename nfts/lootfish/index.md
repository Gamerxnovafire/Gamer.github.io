# LootFish

LootFish 是 Avalanche C 链上的一个独特的 NFT 集合，包含 10000 条美丽而异国情调的鱼。该系列中有数百种外来鱼类，从彩虹鱼到垂钓者到倒钩。 LootFish 涵盖了海洋生物的整个范围，没有两个完全相同。抓鱼并烧掉它们以赚取 AVAX，或者在装饰精美的迷你水族馆中饲养它们！
