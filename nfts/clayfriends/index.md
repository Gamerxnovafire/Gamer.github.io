# Clay Friends

世界著名的定格动画师 Trent Shy 很高兴推出 Clay Friends，这是一个由 100% 真正粘土制成的 5000 件生成系列。 Clay Friends 将精心手工制作的艺术作品与独一无二的社区体验相结合，代表了 web3 和世界级艺术的统一。

什么是粘土朋友？
