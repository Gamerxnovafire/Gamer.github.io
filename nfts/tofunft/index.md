# tofuNFT

什么是豆腐NFT？
NFT 市场专注于 GameFi 和收藏品。 tofuNFT 是一个功能齐全的去中心化市场，用于购买、销售和交易 NFT，由 SCV.Finance 创建并部署在多个区块链上。

TofuNFT — 第一个 Omnichain NFT 交易所
