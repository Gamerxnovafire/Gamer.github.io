# Wirtual

WIRTUAL 是一个“锻炼赚钱”的应用程序。锻炼，获得奖励——就这么简单！通过锻炼赚取 WIRTUAL 硬币，并将硬币兑换成商店购买的实物产品和 NFT、活动门票、组织虚拟体育赛事和持有者排名以访问功能。
WIRTUAL 为其用户提倡积极的生活方式。我们在跑步、步行、游泳、骑自行车、锻炼和舞蹈方面创造挑战，我们的用户可以在速度、距离方面进行竞争，作为一个群体进行比赛，甚至创建自己的自定义挑战。
