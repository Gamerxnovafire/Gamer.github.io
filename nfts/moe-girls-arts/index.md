# Moe Girls Arts

Moe Girls Arts是由人工智能生成并精心挑选的动漫女孩NFT的NFT合集。该集合由两组 NFT 组成。收藏家套装（#00001 至 #01000）：该套装中的 NFT 较为稀有，数量有限，数量为 2.5K。每个 NFT 都包含一个命名角色，并带有 20 种不同的背景。稀有度从普通 (C)、稀有 (R)、超稀有 (SR) 到超稀有 (UR) 不等。由于供应有限，它们被设计为增值资产和长期价值储存，最适合收藏家和投资者。标准集（从 #01001 开始）：该集中的 NFT 更常见，但价格更低。它们被创建为每个人都可以轻松获得，从而使人们更容易成为持有者并加入社区。
