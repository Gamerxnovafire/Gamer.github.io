# Mollector Original Artworks

欢迎独家访问和拥有 Mollector Game 在 Mollector Card 游戏中看到的游戏艺术和概念艺术原创作品。每件艺术品仅铸造一次，因为持有者享有艺术品所有权之

外的独家特权，包括：

1、 在持有 NFT 的同时，终身优先访问所有 Mollector 游戏活动和社区活动。

2、 Mollector Game 生态系统中的终身 VIP 特权，包括持有 NFT 时的勇士令状奖励。

3、 通过私人 Disc·ord 频道直接访问开发者团队。

4、持有者贡献的所有创作者收入将用于 Mollector 游戏社区的福利。

Mollector 是一款 NFT 可收藏的自动战斗纸牌游戏，为 Blockchain on Blockchain 从头开始构建。该游戏提供了一种基于套牌构建、策略和即玩即赚方法的身临

其境的体验，将界限推向“玩即享-致-探索”。
