# Trend Pass

效用
Trendpass是Opensea使用Opensea Api 授权的专用软件。使用Trendpass，您可以在同一屏幕上监控所有活动。顺应潮流从未如此简单。使用Trendpass，您可以快速了解用户现在在做什么，并尽早决定您的位置。一起看所有活动，跟风！是的，仅此而已。

路线图
使用Trendpass v2，我们将使用人工智能自动过滤整个数据流。我们将向您转发整个流的特别通知和诈骗分数。让我们举个例子。例如，如果用户多次铸造同一个集合，该集合的骗局分数就会增加。如果有太多不同的用户铸造，诈骗分数将会下降。拥有大量优质收藏品的用户如果铸造收藏品，将获得更多骗局。这样，我们将根据收藏品的所有者给收藏品打分并为您过滤。
