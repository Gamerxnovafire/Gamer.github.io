# Maneki Gang

Maneki Gang NFT Collection 是一款高质量的个人资料图片艺术，具有 180 多个手绘特征，分布在 8 层中，具有无与伦比的多样性和独特性。这些精美的艺术品以日本招财猫为蓝本，以 ERC-721 代币的形式存在于以太坊区块链上。除了完全授予代币持有者的高分辨率艺术品版权外，每个 Maneki Gang 还可以访问 Lucky Maneki 网站和 Discord 上的福利。

我们是一个去中心化的自治组织，是一个完全由社区驱动的项目，专注于提供涵盖整个范围的产品套件，从付费玩到赚取游戏，再到通过高端商品（酒、服装等）授权我们的知识产权。
