# Lucky Tags Armada Genesis Collection

幸运标签 Armada Genesis Collection 是 Jenkins2d 的 Genesis NFT 系列，由 500 个生成的 Kaijus 组成，由手绘特征制成。从速写本到元宇宙！

由 Jenkins2D 创建的 5,555 个随机生成的 Kaijus，幸运标签舰队 (LTA) 从 5,555 个随机生成的 Kaijus 开始，由 Jenkins2D 赋予生命。Kaijus 是您第一次有机会进入 Jenkins2D 和 LTA Metaverse 的脑海！

每个角色都是我内心感受的延伸。这些角色体现了活着的不同方面。被爱，被恨。
