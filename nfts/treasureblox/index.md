# TreasureBlox

什么是TreasureBlox？
TreasureBlox 是世界上第一个加密和现实世界的寻宝游戏。玩家使用 BLOX 进入寻宝游戏，您可以在其中找到线索并解决谜语以赢得宝藏。

然后他们打开 DAPP 并选择他们想进入的寻宝活动。然后玩家在 BLOX 中支付一次性费用进入狩猎。

进入狩猎后玩家可以申请时间锁定。时间锁定是一个 90 秒的时段，玩家可以在其中输入谜语的答案 - 在此期间没有其他玩家可以尝试获胜。实施时间锁定是为了防止机器人窃取玩家的答案并抢占先机，将他们击败到宝箱。

如果玩家输入错误的答案，他们仍然可以在时间锁定处于活动状态时尝试解谜。如果玩家输入正确答案 - 恭喜！你中了宝罐！本次寻宝活动现已结束 - 但没有获胜的人，请不要担心！有许多新的、令人兴奋的狩猎正在筹备中。
