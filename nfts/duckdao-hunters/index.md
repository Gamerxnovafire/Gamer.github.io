# DuckDAO Hunters

DuckDAO Hunters 是一款可收藏的纸牌游戏，参与者竞相完成 NFT 集，以便有机会在赛季中赢取奖品。
游戏由 48 张卡片组成——每张卡片都有自己的稀有度、故事和风格。 大奖将授予收集所有 48 张卡片的人或在赛季结束时拥有最多卡片的人。
