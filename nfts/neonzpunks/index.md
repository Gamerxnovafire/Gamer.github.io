# NeonzPunks

过去 7 天没有售出 NeonzPunks。

NeonzPunk 是 CryptoPunks 的霓虹灯版本。CryptoPunks 于 2017 年年中作为固定的 10,000 个项目推出，并成为 ERC-721 标准的灵感来源之一。他们曾在纽约时报、伦敦佳士得、迈阿密巴塞尔艺术展和 PBS NewsHour 等地出现。

NeonzPunks NFT - 常见问题（FAQ）

▶ 有多少 NeonzPunks 代币？
