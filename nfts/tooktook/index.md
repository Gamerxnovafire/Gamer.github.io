# TOOKTOOK

TOOKTOOK是一个三秒的日常问候平台，为所有人的日常生活进行创意交流，视觉消息平台。 它不是使用现有的 SNS 或 Messenger 平台提供的无论用户偏好如何的表情符号，而是允许您使用自己的短视频和自己的文本进行交流的平台。 在全球 SNS 市场，超过 125 亿的冗余用户平均每天发送 100 亿个表情符号。 TOOKTOOK 平台允许任何人用自己的 3 秒视频和语言分享他们的日常生活。 您可以创建一个 TTcon 来代替标准化的表情符号来使用。 特别是 TOOKTOOK 的 3 秒视频是一个差异化因素，因为它可以通过个性化和所有社交网络进行交流。 从技术角度来说，我们提供了3秒视频创作的3秒编辑方法。
