# Robbie Trevino - the return

罗比的作品将熟悉与超现实和意想不到的结合在一起。 机械与生物。 他独特的审美风格在许多产品和游戏中都有体现。 他在 2019 年巡演中为 Tool 乐队创作了海报作品，并在过去 9 年中为 Magic the Gathering 创作了许多心爱的卡片。 他还参与了爱情、死亡和机器人以及其他 Netflix 剧集和电影的制作。 他目前正在与著名的电子艺术家 DeadMau5 和 Aether Audio 合作。 所有 4 个 Gateway 作品和 The Elder ee 作品都包含 Aether Audio (Jason Taylor) 的音乐
