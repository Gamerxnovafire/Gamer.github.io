# DLIKE

<p>DLIKE 是一个社交媒体 dApp，您可以在其中获得分享奖励！ &nbsp;</p>Dlike 是简单易用的 Steem 应用，基于你在网络上找到的想要分享的宝藏。 你只需提交网址，Dlike 会抓取该网址的数据，比如图片、标题、描述等，即可完成提交。你可以向主流社交媒体网站分享这些网址。当其他人喜爱你的喜爱时，你可以获得 STEEM 或者原生 Dlike 代币奖励。
