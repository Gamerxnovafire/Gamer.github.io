# Toktikk To Earn

Toktikk 要赚取什么？
Toktikk To Earn 旨在成为一个多功能的视频录制应用程序，主要使用短视频 - 多利润产品结合 Web 3.0 技术，帮助用户通过使用 TokTikk To Earn App 赚取 $T2E 来赚钱。

该项目为用户提供了 TokTikk To Earn App，用户可以通过它使用 $T2E 录制他们喜欢的主题的视频，观众可以查看、评论和与您的视频互动。

我们如何从 Toktikk To Earn 中获得价值？
视频录制 Toktikk 赚取
完成每日任务
排名靠前、顶级、活跃度最高的有机会获得稀有NFT
出售/出租 NFT CD 和 NFT VR 眼镜
幸运轮盘：完成当日任务时，用户将获得 1 次幸运旋转
参加按主题组织的竞赛和活动
