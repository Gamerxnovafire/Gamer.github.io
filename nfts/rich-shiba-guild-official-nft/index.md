# Rich Shiba Guild OfficiaI

Rich Shiba Guild 是托管在以太坊区块链上的 5,555 个 NFT 的集合。

每个 Rich Shiba 都是独一无二的，是一件独特而详细的 3D 艺术品，基于 600 多个元素，灵感来自于 Shiba Inu 狗的财务乐观主义。

成为 Rich Shiba Guild 的一员意味着金融繁荣、第一个准备好 Metaverse 的公会以及彻底改变 NFT 的概念。

我们的主要目标之一是在在线和现实世界中创建一个强大而团结的 Rich Shibas 社区。我们社区的繁荣意味着项目的成功销售。

这就是为什么我们计划为 HOLDERS 的项目创建一个30,000 美元的集合钱包，以确保其寿命。
