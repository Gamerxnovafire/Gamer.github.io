# CryptoToysClubNFT

1.Crypto Toys Club是做什么的？
帮助音乐人、漫画家、电影制作人推出自己的 NFT 并以权利和粉丝 DAO 的形式从中获利。

2. Crypto Toys Club DAO 会做什么？
为音乐家、漫画家和电影制作人拍摄音乐视频、录制音乐和制作作品提供财政支持。

3. Crypto Toys Club NFT 和 Plath DAO 是什么关系？
就像 BAYC 和 Yugalabs
