# Cat and Mouse Game (Official)

猫捉老鼠的游戏正在进行中！在虚拟世界的农村死水里有一个庞大的乡村牧场。里面住着疯狂的猫女士，一位神秘的老妇人，对陈年精美的$CHEDDAR有着巨大的胃口。

但她家的墙壁上隐藏着一个可怕的秘密——密西西比河这一边最大的老鼠家族！老鼠们通过离开安全的墙壁去觅食$CHEDDAR来维持它们的数量。当老鼠有足够的食物和住的地方......好吧，我们可以说它们可以制造更多的老鼠。

但 Crazy Cat Lady 的奶酪储藏室并非毫无防备！要成为疯狂的猫女士，您需要很多猫，它们会成为真正的切达干酪守护者。每当老鼠抓到的东西超过了它们所能携带的东西时，猫就准备好把掉下来的东西抢回来。如果小老鼠小偷不够小心，他们会在他们试图回家时被杀死。
