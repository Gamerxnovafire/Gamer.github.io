# Ludo Labs Genesis Collection

Ludo Labs 是一个基于会员制的数字足球社区，利用 NFT 的力量为球迷提供独家体验，并以民主化的方式进入足球核心圈。

通过拥有我们的 NFT，收藏家成为全球志同道合的球迷社区的一部分，他们对足球充满热情，同时也有可能直接从包括保罗·迪巴拉在内的一些世界上最好的足球运动员那里赢得奖品、互动和体验， Edin Dzeko、Diogo Dalot、Miralem Pjanic、Patrice Evra、Mohamed Elneny 等。

过去 7 天内没有售出 Ludo Labs Genesis Collection。
