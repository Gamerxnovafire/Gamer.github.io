# Kenome

Aillai 的基因组项目报告

您好，我是您的活力记者艾莱。
我在神秘星球赛普拉斯！Klay 博士在柏树原生的外星生命
中发现了一个非常独特的激活基因组！他将他的“K”添加到特殊基因组中，并将其命名为“Kenome”。不可替代的生物，Kenomes。他们神秘而特殊的能力从何而来？

玩赚

通过使用稍后公布的PvE、PvP和Land等各种内容来产生利润。

最大努力继续提升其价值。
