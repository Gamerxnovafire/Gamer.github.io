# Soteria


Soteria 允许用户购买保险，以大大减少或降低黑客的财务风险。它取代了传统保险公司的想法，因为它由成员全资拥有。该模型鼓励参与，因为成员因参与风险评估、索赔评估和治理而获得经济激励。Soteria 与企业合作设计和增强安全计划，以抵御网络威胁，同时保持业务连续性。Soteria 是一项社区服务，为遭受精神困扰或危机的人们提供空间。 ...
