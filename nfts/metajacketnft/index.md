# Meta Jacket NFT Official

META JACKET NFT 是由艺术家和时装设计师 Sheycha 设计的 888 件夹克的集合。Meta Jacket NFT 的持有者有机会将他们的 NFT 变成真正的 Jacket，并从永久销售中获得版税。

多年来，我们的团队一直在制作来自不丹王国的定制手工夹克。这个系列让我们有机会展示不丹丰富的传统工艺，将现代时尚与NFT 的无限可能性完美结合。我们想在 web3 中重新构想时尚的未来外观和感觉。随着时间的推移，我们希望分散创作过程，并与我们所有的 NFT 持有者分享未来合作伙伴关系的好处。
