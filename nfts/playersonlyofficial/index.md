# Players Only (Official)

Players Only NFT 由职业运动员创立，汇集了 4,995 名独特的篮球、棒球和足球运动员，他们生活在以太坊区块链上。每项运动由 5 支球队组成——银背队、天鹰队、鲨鱼队、龙队和眼镜蛇队——每支球队都将在虚拟锦标赛甚至现实生活中相互对抗。
我们将为所有持有者提供赠品作为实用工具。每周都会选出获奖者。在选出另一名获胜者之前，获胜者有 24 小时的时间申领。这将在钱包验证后在 Discord 中完成。

不过，我们的重点是 IRL 体验，因此这些 NFT 将反映这一点。无论是一对一的游戏、抽搐游戏流、练习设施之旅、与伙计们一起观看 UFC 比赛、为比赛挑选他们穿的鞋子等等。我们有一些非常有趣和独特的你们很快就会听到的经验！
