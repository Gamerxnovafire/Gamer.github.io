# ToBet

什么是ToBet？
ToBet 是一个基于区块链的去中心化游戏平台。 平台推出了许多有趣的游戏：骰子、水果游戏、二十一点、撞车、三公爵和快乐公牛。 更多有趣的游戏将在稍后推出。 我们的目标是创建一个公平、公正和开放的游戏生态系统，并汲取人们喜爱在线游戏的精髓——刺激和兴奋，以及游戏的社交方面。
