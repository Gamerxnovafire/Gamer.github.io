# Meebits

Meebits 是 20,000 个独特的 3D 体素字符，由自定义生成算法创建，然后在以太坊区块链上注册。管理所有权的 NFT 合约是标准 ERC-721，适用于任何兼容的服务或交易所。合同中还包括一个自定义市场，支持每笔交易最多 100 个 Meebits 的同类交易，以及所有标准的买入、出价和卖出交易。Meebits NFT 在过去 7 天内售出 50 次。Meebits 的总销售额为 286.33 万美元。一个 Meebits NFT 的平均价格为 5.7 万美元。共有 6,581 名 Meebits 所有者，总供应量为 20,000 个代币。
