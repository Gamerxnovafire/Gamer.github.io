# YOLO

YOLO是一个分散的交易所，旨在在EOS网络上提供无摩擦的令牌交换体验。它是为每个人准备的 — 从新手到交易忍者。在YOLO上交换代币非常简单（无订单簿，无中间代币），即时（在EOS和可用流动性储备上运行），并且安全（无需存款；从您自己的钱包进行交易）。YOLO将Kyber久经考验的流动性协议引入EOS，其明确目标是提高基于EOS的项目的流动性和可用性，最终也提高基于ETH的项目的可用性。除了DEX提供的功能外，还有多个新功能：速度、简单、无存款、优化的价格发现、跨链功能，而且别忘了，它得到了Kyber网络团队的支持。
