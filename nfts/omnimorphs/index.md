# Omnimorphs

Omnimorphs 是一个生成式 NFT 艺术收藏项目，包含 10.000 个独特的数字手绘头像，在以太坊区块链上永生。除了令人惊叹的优质艺术品外，我们还致力于为代币创建额外的功能，例如 OmniFusion、未来的空投、可认领的插件收藏等等。

Omnimorphs 是地球上的模拟神秘实体，能够通过OmniFusion融合和进化。Fusion 实用程序使所有者有机会将他们的两个代币组合起来，以自定义其头像的外观和稀有度。NFT 系列最初由数字艺术家Dániel Taylor创作的 10,000 件手工艺术品组成 。随着每个 Fusion 燃烧一个 Omnimorph，这个数字会减少，作为回报，它的主人会得到一个灵魂碎片。
