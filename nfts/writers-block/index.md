# Writers Block

**作家块**

Writer's Block 讲述了一个找不到自己声音的小孩的故事。他患有语言障碍，说话时结结巴巴。同学们都笑了，他也就沉默了。因不完美而愤怒，他带着痛苦走进去。他感到不断的羞耻，他用咬来惩罚自己。很快，齿痕就无法切得足够深，所以他选择了刀片。

他不想死，只想逃避今生的伤痛。最终，他用写作代替了切割，将他的感受变成了文字，尽管有时他的句子无法捕捉到他的焦虑。在他最黑暗的时刻，他转向接近死亡，但从未放弃对上帝的希望。

在出版商拒绝了他的作品后，他停止了写作，全身心地投入到油画棒中。这个集合被称为 Writer's Block，创建于 1998-1999 年，就是结果。尽管有许多障碍，他还是更加致力于他的手艺。不久之后，他找到了他的灵魂伴侣，他们结婚了。如今，他是畅销书作家和出版商。
