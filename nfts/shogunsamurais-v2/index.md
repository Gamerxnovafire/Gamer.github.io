# ShogunSamurais

武士道八法典
武士道（武士道，“武士之道”）是与侍（武士）态度、行为和生活方式有关的道德准则，所有武士都在此生死存亡。

武士道法典的这 8 种美德分别由 8 位传奇的将军（将军，shogun）体现：

吉：“正义将军”，武士对他们的决定做出充分的承诺。

于：《勇气将军》，英勇的勇气不是盲目的，而是聪明而坚强的。

斤：《慈悲将军》，武士发展出一种可以而且必须被善用的力量。

丽：“尊敬的将军”，武士没有理由残忍。

诚：“诚信将军”，说话就是做事。 武士不必言出必行。

明阳：“荣誉将军”，武士的行为反映了他们的价值。

中木：《使命的将军》，对于他们负责的每个人，武士仍然非常真实。

智成：《克制将军》，对于武士来说，征服自己是所有胜利中的第一个也是最崇高的。
