# The Hooligans by ANOMSTER

Hooligans 是 4,000 个独特的 1/1 游戏化身的生成集合。 流氓是侵入以太坊区块链的虚拟朋克。 受到 90 年代早期复古游戏的启发，Hooligans 将成为您可以在 Hooliverse 和其他基于以太坊的侧向滚动游戏中使用的第一个 2d 侧向滚动游戏角色（如果所有 4,000 个流氓都被铸造）。 这一系列 ragtags 包括一组不同的男性和女性，有 7 种不同的皮肤变化。 每个 Hooligan 都有超过 600 种独特的特征，如头发、胡须、配饰、伤疤、服装等等。 除了物理属性外，每个角色还将拥有单独的游戏相关统计数据，例如 HP、XP、跳跃力等。 来拿些吧！
