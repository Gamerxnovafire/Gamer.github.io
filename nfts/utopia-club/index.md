# Utopia Club

Utopia NFT 俱乐部是一个紧密结合的独家社区，是印度尼西亚 NFT 领域具有不同背景的有影响力人士的大熔炉：从低级到投资者，从名人到企业家，以及连接传统品牌和全球 web3 项目的 web2 到 web3 专业人士。  

拥抱权力下放的本质，我们在合力制管理中运作，其中每个工作组都有一个自组织系统和由具有非等级、共识和集体主义价值观的激进主义做出的决定。

UTOPIA FAMILY 对我们来说是一个全职项目，我们已经计划了多年的实用程序，不断为 UTOPIA FAMILY NFT 持有者提供更多价值。每个季度我们都会推出新的短期路线图。虽然我们在路线图上有很多计划，但我们还计划随机发布奖金公告，为我们的长期持有者带来更多价值。
