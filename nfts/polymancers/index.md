# Polymancers

Polymancers从奥术和元素力量中汲取力量。存在四个团体，每个团体都专注于某种魔法力量。这些团体通过斗篷的颜色表示他们的忠诚。然而，一些 Polymancers 拒绝与这些群体结盟。
