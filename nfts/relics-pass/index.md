# Relics Pass

Relics Pass 由IKONICK& Sneaks of Nature的联合创始人 Jeff Cole 和 Mark Brazil 带给您。Pass 是 IKONICK 的核心

持有 Relics Pass 将授予您一个免费薄荷和一个折扣薄荷到 Sneaks of Nature，这是科尔心目中的特许经营权。

Relic Reserves：1000 多个单位收藏，免费或以极低的价格提供给我们的 Relics Pass 持有者。这些以每年至少两次的速度发生，由 Jeff Cole 创建或策划。Relic Reserve 系列使我们能够与出色的艺术家合作并探索 NFT 的技术创新。

验证您在 Discord 中的通行证以查看 Relic Room、与顶级企业家一起参加演讲者系列，或者只是与工作人员一起闲逛。一定要在进来的路上选择你最喜欢的流行挞口味。
