# STONED PIXEL HUMAN V2

Stoned Pixel Human 是一个雄心勃勃的项目，不仅涉及像素化身，还涉及社区，该社区正在建设成为 Metaverse with Music 的一部分。
SPH 由聪明的外星人创立。普罗米修斯先驱者是超多样化的，他们来自不同的种族和其他行星，包括地球。

“音乐让你失去控制”
外星人目睹了一种叫做“音乐”的神秘现象，当他们试图接触地球时，当他们看到人们聚集在周围，有说有笑有哭泣，有时还疯狂地跳舞，他们就想研究和探索这种现象。
为了研究被称为“音乐”的神秘空气振动现象，他们招募了研究人员和专家，并得以在社区内发展壮大。
这个社区的早期参与者戴着像素面具，谈论音乐、宇宙和未来。

这是 Stoned Pixel Human - SPH 的开始和崛起。
