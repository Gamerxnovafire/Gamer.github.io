# LUVRworldwide

alissachristine.love 的 XR/VR/AR 治疗艺术

Alissa Christine，LUVRworldwide 的创始人，是一位多维、多感官、高振动、有远见、虚拟现实、治愈艺术家、摄影师、XR 体验设计师、作家和加密策展人。她的表现专注于通过用爱激发内在成长、灵性和可持续性来实现积极的改变。

Alissa 是一位拥有 20 多年经验的迈阿密混合媒体展览艺术家，被公认为 VR 先驱，在 2 次 Live Global VR 绘画比赛中获得第一名。她曾参加过首届 WoCA（Women of Crypto Art）展览，并在 AltspaceVR 制作了首个 NFT VR Crypto Art Gallery World。
