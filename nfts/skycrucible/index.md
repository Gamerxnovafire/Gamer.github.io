# Sky Crucible

选择一种材料来制造你的武器。然后用强大的球体升级你的武器。我们的锻造经验允许根据您想要的游戏风格定制 Sky Crucible 宇宙中的每件武器。

与大多数 NFT 收藏不同，你在 Sky Crucible 中铸造的每一套都带有五种物品 (NFT)，可以通过多种方式组合。锻造武器、与他人交易组件或在交易所出售您的物品。
