# Msea Platform

**东南亚平台**

Msea将打造GameFi世界第一个游戏道具交易平台。
通过整合传统模型和区块链技术，连接元界世界。

作为为Gamefi世界开发的游戏道具交易平台，Msea的出现将为Gamefi项目增添创意、可玩性和生存能力。
Msea将使用区块链技术打破各个游戏团队之间的壁垒。不仅作为交易平台，Mesa 还将促进游戏开发者之间的合作，帮助 Gamefi 世界的开发者。

Msea.io 和 Opensea 有什么区别？
本质上两者都是交易平台！但 Opensea 是一个独立的 NFT 交易市场。Msea不仅提供NFT交易，还帮助Gamefi行情道具上链交易。
