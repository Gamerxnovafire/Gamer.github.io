# NiftyKit

创建您的合同，上传您的资产，然后在准备好时启动。 每个 NFT 都是惰性铸造的，因此汽油费会转嫁给收集者。 最适合进行较大（例如 100-10,000）次投放的创作者和团队。直接在您的网站上嵌入薄荷按钮，为您的收藏家提供无缝和直观的体验。 DropKit.js 与大多数主要的网站构建器和平台（即 Squarespace、Webflow、WordPress）兼容。NiftyKit 让您在几秒钟内轻松创建您的第一个智能合约和 NFT。 可兑换 NFT 是一种有趣且灵活的方式来吸引您的社区，它会生成一个特殊的兑换链接，以便收藏家可以按需购买。
