# SnarkySharkz Society

这是 5000 年，所有人类都破坏了我们的环境，离开地球居住在其他太阳系。世界现在被洪水淹没，只有水覆盖了一个完全被淹没的蓝色星球的表面。

鲨鱼已经取代人类处于食物链的顶端。这些非凡的鲨鱼生物随着智力的提高、类似人类的特征和行为以及扭曲的风格而进化。

Sharktopia 文明在高级议会长老 Sharkz 的领导下蓬勃发展。

为了避免重蹈人类的覆辙，并确保 Sharktopia 的长期生存，高级委员会创建了 SnarkySharkz 项目。为该项目精心挑选了 702 名 Shark 候选人。
这些勇敢的人冒险进行了大迁徙，前往南半球的新殖民地，被称为 SnarkySharkz' Society,

Generative NFT 项目由 702 个独特的不可替代代币 (NFT) 组成。SnarkySharkz 的社区重点是通过协作和参与的 NFT 社区，为产生真正影响的海洋保护工作带来资金和意识。原始铸币已结束。
