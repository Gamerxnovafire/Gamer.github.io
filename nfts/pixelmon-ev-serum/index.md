# Pixelmon Evolution Serum

游戏
最终，Pixelmon 旨在为所有人（无论是持有者还是非持有者）创造令人惊叹的可玩体验。当我们扩大团队并推进开发时，我们心中有一个目标。创造最好的游戏。
运动鞋和装备
训练师能发挥他们的 Pixelmon 最好的一面，而作为回报，Pixelmon 也能发挥出他们最好的一面。当您穿越游戏世界并与您的 Pixelmon 并肩作战时，训练师就是您的化身，每位训练师都因您选择的武器、盔甲和装备而变得独一无二。
像素兽
像素兽是各种形状、大小和种类的独特生物。在我们的虚拟世界中与您一起旅行，Pixelmon 构成了地球和游戏生态系统的基础。它们可以在游戏中被捕获、收集、训练、玩耍以及作为 NFT 出售和购买。
