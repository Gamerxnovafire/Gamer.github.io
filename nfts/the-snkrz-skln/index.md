# THE SNKRZ LOTTERY

SNKRZ 是一个 Web3 移动和赚取社交健身奖励应用程序，允许用户使用他们的 NFT 进行锻炼并在元宇宙中拥有土地。每个 SNKRZ NFT（或鞋子）都有其设计和能力。能力是性能，运气，耐用性，质量和发烧。限量版或高稀有度 SNKRZ 将有一个名为“BOOST”的附加能力。SNKRZ 用户只需步行、慢跑、跑步、骑自行车和徒步旅行即可获得奖励。 分享您的锻炼数据，实现您的健身目标，玩得开心！$SKZ 是用户在使用应用程序锻炼时获得的游戏代币。获得的奖励取决于用户的 SNKRZ NFT 能力。赚取的 $SKZ 可用于升级、繁殖、修理、参加幸运抽奖等。$FNF 是一种供应有限的治理代币，将在游戏中以更高的级别使用。
