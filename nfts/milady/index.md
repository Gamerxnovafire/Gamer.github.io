# Milady Maker

Milady是一种审美、一种生活方式和一个社区。 我们将不断构思和开发 Milady 生态系统的有趣扩展，支持 Milady 村的所有 Milady NFT 持有者。 此处列出了我们计划立即开始工作的一些初始概念。

Milady NFT 横幅：生成式 NFT 标题照片系列，仅适用于 Milady NFT 所有者。 这些横幅的设计采用了一种新颖的生成算法，其风格与 Milady pfpNFT 的美学相匹配，作为其头像优先设计的扩展，并且可以多种格式和分辨率下载。

Milady VRtube：Miladys 被设计为独立的资产层，因此它们已准备好用于 Live2D 装配，用作 VRtube 化身。 文件生成器将允许它轻松设置——我们将看看是否可以自动化索具本身。
