# SadNyang

Sadnyang NFT 是 Claypad 推出的一种新型 NFT，与现有的 NFT 不同，Sadnyang NFT “自动”将二级交易费用分配给持有者，而不是项目所有者。总共有 10,000 种不同的 Sadnyang NFT，NFT 存在且稀有性并不单独存在。
