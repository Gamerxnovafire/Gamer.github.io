# Treasure Hunt

什么是寻宝？
寻宝游戏是 Polygon Network 旗下的一款以幽灵海盗为主题的游戏。 海盗登船造船，突袭岛屿赚取 USDT。 我们没有自定义令牌。 我们使用 USDT 作为游戏内货币和奖励，以确保我们的游戏不受市场价格的影响。

中午休息玩寻宝游戏！寻宝活动让您每天都有机会赢取彩票奖品，在动画寻宝活动抽奖过程中，看着中奖彩票号码从宝箱中弹出是很有趣的。

你能找到隐藏的宝藏吗？

宝藏隐藏在这个海滩的某个地方，网格线相交（交叉）。

输入坐标以帮助以最少的猜测找到宝藏。交互性为您提供了到达宝藏所需的最短距离（沿着网格线）。
