# Rebellion DAO Rebels

Rebellion DAO 是一项专注于为其原生 REB 货币提供价值和流动性的运动。这是来自 DAO、NFT 和 DeFi 的概念的巅峰之作，它试图创建一个强大且可持续的经济模型，奖励参与网络的每个人。
Rebel NFT 可以通过赞助 Rebellion Citizen 来“激活”

每个“活跃”的 Rebel NFT 每天至少产生 25 个 REB

叛军可以通过 XP 和 REB 升级以增加奖励
