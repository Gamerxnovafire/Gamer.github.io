# Punkachus

关于我们
一个完全由两个对区块链技术的潜力完全震惊的家伙制作的激情项目。 受到其永恒能力的启发，我们决定将我们的朋克化版本引入以太坊区块链，这是一种极其相关的文化现象，现在并将继续成为许多童年的一部分。
我们为您带来了一组最可爱的 3000 个随机生成的以太坊区块链 ERC-721 NFT，由 50 多种可能的属性和 5 种稀有性组合而成，并由去中心化 IPFS 提供支持。
除此之外，我们的主要目标是创建一个熟悉、友好和热情的社区，专注于相互学习，这样您也可以以自己的特殊方式为 NFT 社区做出贡献！ 话虽如此，我们将继续研究和探索提升和推动这个项目的新方法！
