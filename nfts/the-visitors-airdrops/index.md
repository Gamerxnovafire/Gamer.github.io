# The Visitors Airdrops

访客持有者的空投居住地！所有收益将捐赠。2021年8月23日第一次接触；游客来了。他们的使命是迅速而彻底地征服地球。尽管他们拥有卓越的情报和技术，但访客很快就遇到了他们的对手。来自人类垃圾箱的流行文化垃圾永无止境的饮食迅速解除了入侵者的武装。太懒惰和没有头脑来完成他们的使命，游客们现在被诅咒互相分享猫的 GIF 和模因，直到时间结束。为了满足他们对垃圾食品和无用物质（斯宾塞的礼物）的渴望，游客们需要钱……快。不幸的是，他们已经沦为无用的生物，缺乏为社会做出贡献的任何才能或能力。在没有其他选择的情况下，他们选择了最简单、最可靠的就业形式：创建和销售 NFT。
