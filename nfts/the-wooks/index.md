# The Wooks

PixelWooks 是一个 NFT 项目，旨在建立一个现场音乐社区并突破 Phan Art 的界限。所有权是您进入 JamDAO 的门票，它将改变 Metaverse 中现场音乐的表演方式。有 5,000 种独特的 wooks，其中许多您可能知道或至少从其中购买过药物。嘘！我们正在建立一个社区，将实验性音乐会和表演艺术带到 Metaverse。拥有 wook 是您的入场券。因此，无论您是 NFT 猎人还是这是您的第一次 NFT 体验，都快来加入派对吧！DAO 意味着你制定规则。Wook 持有者将可以访问 JamDAO，因此您将帮助决定哪些艺术家表演并提出其他惊人的想法。
