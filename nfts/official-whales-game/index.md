# Whales.Game

通过简单地持有 Whales Game NFT 来收集 KRILL。无需质押/取消质押您的代币，一旦您的 NFT 转移给您，KRILL 就会自动开始收集。

有两种类型的令牌；鲸鱼和渔民。每个渔民每天收集 10,000 只磷虾，但当您索取捕获的鱼时，20% 将分配给所有鲸鱼。

在 Uniswap 上添加 ETH + KRILL 流动性并质押您的 LP 代币以收集额外的 KRILL。所有 KRILL 铸造成本的 25% 以及 1% 的 KRILL 转移费用都发送给了质押者。

收集到的任何 KRILL 都可用于生成更多 NFT，以进一步提高您收集更多 KRILL 的速度。随着更多代币的铸造，产生新代币所需的 KRILL 数量将成倍增加。
