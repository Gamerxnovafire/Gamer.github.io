# PunkBodies

PunkBodies是艺术家设计和机器生成的基于以太坊区块链的身体像素艺术图像，与现有的 CryptoPunks 相得益彰。有 10,000 具尸体，每具尸体都与现存的每一个朋克的类型和肤色相得益彰。使用PunkBodies，您可以组合和组成朋克和身体的新组合，创建一个完整的可铸造化身，我们将其称为Punkster。在展示页面上匹配、混合和铸造任何身体和朋克组合。
