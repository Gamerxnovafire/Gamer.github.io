# TokenWarriors

什么是代币勇士？
TokenWarriors 允许玩家购买、收集、繁殖、出售和对抗你的战士。 培育和销售各种类型的虚拟TokenWarriors或让他们战斗以获得更多经验并获得更高的LVL。 高 TokenWarriors 将是 RAR 且昂贵。

Token Warriors 是一款简单有趣的游戏，它基于以太坊区块链以实现完全透明，并且大部分都是免费的。（不排除 Gas 费用）以及如何处理 Token Warriors？让你的战士战斗以获得更高的水平和更多的经验......它可能有助于赢得你的下一场战斗！此外，您还可以从现有的战士中培育出新的战士。还有一个集成的市场，您可以在其中买卖勇士。交易费：我们将游戏设计为使用尽可能少的 gas 来实现我们的功能。交易成本从 0.02 美元到 0.06 美元不等，以太坊网络上的等待时间为 1-3 分钟。有些功能需要一点额外费用，例如我们的育种功能，使用 0.0025 ETH，但所使用的气体/费用低至 0.02 美元到 0.06 美元！
