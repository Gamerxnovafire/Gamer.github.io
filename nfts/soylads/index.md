# Soylads

作为 Yona 和 Cum Box Collective 的骄傲产品，soylad maker 系列是一个卡通风格的作品，展现了对复杂 NFT 艺术的荒谬和痴迷。

每件作品都是由鼠标粗心绘制的，没有任何绘图板（（不是因为作为一名艺术家很糟糕，我希望我对自己的职业更有责任感，但无论如何我在这里画大豆））以确保社区和艺术之间的流畅动态。
