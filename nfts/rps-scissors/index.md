# RPS Scissors

RPS 的使命是为 web3 社区带来公正的决策权，特别是通过向社会提供 web3 分布式预言机服务。我们希望以最简单的方式做到这一点，以流行的游戏剪刀石头布👊✋✌️为例。


借助 RPS 分布式预言机服务 DApps🔮，任何社区、DAO、体育游戏或赌博应用程序都将能够以公正的方式解决链中的争议。由于 RPS 做出的任何决定都将存储在区块链（PoS 区块链）中，任何人都可以请求证据结果以供将来证明。

RPS 是 3 个 drop 的多集合，包含在 Ethereum ERC-721 合约上创建的10k 实用程序 NFTS。RPS 成员将能够参加竞赛，使用他们的收藏品在链上记录决策，并且每当 RPS 预言机分布式服务被元界中的其他社区或 DApp 使用时，也将受益于按使用付费。
