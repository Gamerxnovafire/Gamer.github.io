# KnuckleHeadz

什么是 Knuckleheadz？KnuckleHeadz是Ruben “Dario” Ramirez的心血结晶，也是Animus Collective下的第一个PFP项目。该项目的灵感来自纽约市标志性和充满活力的人们，是对城市/内城文化和人的庆祝。该项目的核心是提升和支持市中心社区的组织和小型企业的愿望。我们的目标不仅仅是做另一个可收藏的项目，而是创造一种方法来帮助人们和运动对服务不足的社区产生积极影响。我们努力构建一个看起来很酷的项目，并且在代表您在城市周围看到的许多面孔方面做得很好。达里奥和阿尼穆斯都不想把我们的名字放在匆忙、不假思索的工作后面。因此，我们正在尽最大努力确保该项目拥有我们所能聚集的最高质量的艺术品。
