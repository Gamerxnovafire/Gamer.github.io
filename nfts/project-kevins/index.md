# Project Kevins

▶ 什么是凯文斯项目？
Project Kevins 是一个 NFT（不可替代代币）集合。存储在区块链上的数字艺术品集合。
▶ 有多少 Project Kevins 代币？
总共有 3,000 个 Kevins NFT 项目。目前，383 位所有者的钱包中至少有一个 Project Kevins NTF。
▶ Project Kevins 最昂贵的交易是什么？
售出的最昂贵的 Project Kevins NFT 是 Kevin #2513。它于 2022-06-11（3 个月前）以 5.6 美元的价格售出。
▶ 最近卖出了多少个 Project Kevins？
过去 30 天内售出了 2 个 Project Kevins NFT。
▶ 什么是流行的 Project Kevins 替代品？
许多拥有 Project Kevins NFT 的用户还拥有 GasWarriors Clones、 Invisible Society XYZ、 EL NUMEROS和 DegenOkayBears。
