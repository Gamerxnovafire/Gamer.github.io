# The Heartbreakers

当地科学家 Rachel Brains 想出了一种新的肥沃作物肥料。本发明没有达到应有的效果，并且测试显示了副作用。市议会不会允许制作它。一位黑手党老大找到了这位科学家，他对这项发明有自己的阴险计划。他提出资助进一步测试该公式并为他工作。黑手党想要占领这座城市，所以他们需要转移当地警察和市政府的注意力。黑手党在农业方面占有一席之地。他们开始给水果和蔬菜施肥。食物最终出现在商店货架上，然后送到常住居民手中。居民开始变异，一些HB开始长出尖牙和红眼……市长阿纳斯塔西娅·哈特和警察局长伊丽莎白·贾斯蒂斯发表声明，称该市正在发生奇怪的事情，并警告市民要格外小心。
