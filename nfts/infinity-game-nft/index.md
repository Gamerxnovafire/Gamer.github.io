# Infinity Game NFT

什么是Infinity Game NFT？

(IGN), Governance Token Infinity game NFT 是一个 NFTs 游戏平台，以 IGN 作为其治理代币。我们的目标是为我们的用户提供种类最多的游戏，同时不放弃良好的乐趣和盈利能力。
我们有一个完全专注于创建和开发游戏的团队。我们正在为 NFT 市场带来多项创新。
IGN与币安智能链（BSC）网络相连，属于币安智能合约的创新模式。
