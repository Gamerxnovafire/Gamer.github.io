# TradingLab

什么是交易实验室？
TradingLab 的核心是投资组合/投资跟踪器。

专注于清晰和最小的设计，它允许无缝监控多个加密投资。

使用 TradingLab，用户始终能够以最小的努力了解投资的状态。 功能集是有限的，在学习过程中创造了较低的进入门槛。

根据设计，每个用户都有一个与区块链地址相关联的唯一 URL。 该 URL 可以由任何人共享和打开，除非用户另有决定。 这种开放性允许创建“无限滚动体验”，其中一眼就可以看到其他用户在加密投资方面做了什么。 用户可以滚动浏览提要，跳转到特别有趣的交易者资料。 在探索了宏观投资组合之后，用户可以打开单笔交易以可视化详细信息、添加评论并可能关注交易者。
