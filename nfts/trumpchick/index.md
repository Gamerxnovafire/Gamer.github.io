# LoserChick-TrumpChick

一枪
规则
您可以选择多个 Trump NFT 参与竞选。
3 个 Trump NFT 将从所有参与的 Trump NFT 中随机抽取。
这 3 个 Trump NFT 的所有者将成为获胜者（每个地址一次只能获得一个奖品）。
获胜者将获得一个全新的神秘小鸡头像。
所有参与的 NFT 将在活动结束后被销毁，等待下一批神秘小鸡的到来。
炼金术
规则
-
玩家可以通过将其地址中的一个或多个 NFT 放入炼金池中来参与活动。
-
最终，单个地址入池的所有 NFT 中，总分最高的前 4 名获胜。（如果有相同分数的，则按照入池时间排序。）
-
NFT的分数=基础分数+属性加分+附属加分。
-
获奖者可以在宣布奖品后合成他们的头像。
