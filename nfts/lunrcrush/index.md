# Lunrcrush

LunarCrush 收集社交媒体上有关比特币的活动、数以千计的 alt 和交易所，以及有影响力的人发布的内容，并将其提炼成易于消化的小块。
您使用 LunarCrush 的次数越多，它就会变得越聪明，了解您感兴趣或不感兴趣的内容。通过将收藏夹、馆藏和提醒添加到您的帐户，您将获得对您最重要的社交数据。
