# Rengoku Legends [Spiritkeepers]

氏族:
在自定义过程中,持有者可以在几个氏族之间进行选择。武士的氏族决定 了可以选择的设备,以及未来阶段的未来对齐方式。要了解更多关于氏族 及其背景的信息,请访问 Discord 服务器并查找#clan-list 频道。
背景:
这是武士的背景图形,在自定义过程中会自动滚动。
核心槽位:
每个武士至少提供四个槽位,分别是头盔、胸部、面板和遮阳板槽位。这 些插槽可以具有基于随机化(滚动)的常见、稀有或传奇选项。
附加插槽:
每个武士都有随机机会获得最多两个额外的插槽,即翅膀插槽和装饰品插 槽。武士可能没有,其中一个或两个插槽,这也可以是常见的,稀有的或 传奇的选项。
这些属性会影响武士的稀有度,我们已经与Rarity.Tools帮助为感兴趣的 人提供收藏中的稀有度排名。武士的稀有性目前主要是为了收藏家的目 的,对武士的实用性没有任何影响。
