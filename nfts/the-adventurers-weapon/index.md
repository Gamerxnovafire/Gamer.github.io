# The Adventurers Weapon

冒险者不是另一个没有长期实用性的基于文本的一次性 Lootproject 克隆。我们了解文本 RPG 主题 NFT 背后的乐趣和吸引力，但我们也了解 NFT 空间非常注重艺术。因此，我们受到启发，将两全其美的优点结合起来。幸运的铸币者在我们的隐形下降期间设法抓住了我们的一个免费 NFT 将通过持有他们的 NFT 受益匪浅，因为我们将在第一周内开始推出实用程序。

我们的计划是通过我们最初的免费 NFT 投放来降低进入门槛并增加采用率，并利用收益为 NFT 社区带来实用性，并继续建立一个有趣和令人兴奋的社区。这就是为什么我们决定完全免费公开发布我们的第一滴！
