# Empow Social Network

EM 币是 Emow Blockchain 的原生币。与其他区块链网络的其他原生币一样，EM 币是 Emow Blockchain 的燃料，是鼓励区块生产者开展工作的激励计划的重要组成部分。特别是在 Emow 系统中，用户还可以通过在 Empow 平台上贡献有价值的内容来赚取 EM。
