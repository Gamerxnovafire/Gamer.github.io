# Writer's Dorm (Matic Live Test)

阅读引人入胜的故事的体验不应该简单地在最后一行结束。当我们完成一个伟大的故事时，我们多久感到迷茫和空虚？当我们需要回答有关人物和情节的紧迫问题时，或者当我们想与志同道合的人谈论故事可以或应该如何演变时，我们会怎么做？

作家宿舍是作家和读者联系的新方式。读者现在可以完全沉浸在一个故事中，并通过对话、交替的情节结局、角色塑造、粉丝艺术以及您的想象力所能想到的一切来满足他们对该故事的依恋。**我们是故事结束时开始的社区。**

开始使用 Writers Dorm 的最佳方式是加入我们的[Discord 频道]，您可以在其中了解有关我们社区的更多信息。

\1. 要进入作家宿舍，您需要[购买故事的数字通行证。]

\2. 获得数字通行证后，返回 Discord，我们将向您展示如何使用通行证进入作家宿舍。

\3. 与您的作家和读者社区互动愉快。
