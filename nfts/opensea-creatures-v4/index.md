# Dig Collections

dig 查找针对 DNS 服务器运行查询以检索特定名称（FQDN - 完全限定域名）的 DNS 记录。可以以这种方式查找任何 DNS 记录。

有几种不同的语法可用于指定应检索的记录以及名称。也可以明确指定用于查找的 DNS 服务器。
