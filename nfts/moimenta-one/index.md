# Moimenta One

这是 Moimenta One 的故事，它是一艘雄心壮志和规模庞大的一代飞船，使用的技术超出了殖民者的理解 - 以及 Moimenta One 正在向一个密集而神秘的轨道世

界提供的分层社会中的数千万人的故事由一个难以置信的大月亮，就像一个保龄球的浮石在一块铅周围滚动。

![Moimenta One](Moimenta-One01.jpg)

穿越到 587.1 光年深空的另一边，殖民者很快就会发现，距离的屏障是他们抵御等待他们的未知恐怖的唯一防御。当他们陷入人类的偏见和阴谋中时，认识到他们

已经觉醒的危险将比他们在数千年的旅程中面临的任何挑战都更具生存必要性。
