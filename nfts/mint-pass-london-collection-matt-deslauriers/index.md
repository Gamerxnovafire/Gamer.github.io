# Mint Pass London Collection | Matt DesLauriers

The London Collection 是[Bright Moments](https://brightmoments.io/london)精选的六位领先数字艺术家，在伦敦市中心铸造 IRL。每位艺术家都创造了一种身临其境的、独特的、模拟的风格来为他们的收藏制作输出。

薄荷通行证持有人有权从艺术家的伦敦收藏系列中获得铸造的 NFT，包括在 7 月份访问 13 Soho Square 以铸造 IRL 的机会。亲自铸币将使您能够见到艺术家，拥有与铸币厂相关的独特记忆，并共同创作作品。如果铸币通行证持有人无法在铸币窗口期间访问伦敦，他们也可以远程铸币或通过代表铸币。

关于艺术家：Matt DesLauriers 是一位居住在伦敦的程序员和生成艺术家。他使用算法为印刷、网络和装置创作艺术。
