# reNFT - Genesis Cards

我们是一个多链 NFT 租赁协议和平台，使项目能够将 NFT 租赁功能快速集成到各自的项目中。

专为 Web3 领域的开发人员而构建，我们的目标是通过允许项目将我们的协议无缝集成到他们的项目中而无需更改其前端——这也称为白标，从而使项目的生活更轻松。

此外，项目还可以简单地使用reNFT 市场上的，我们支持三种不同的租赁解决方案：预付款、奖励分享（即将推出）和抵押租赁。称为传统租金，通过将 NFT 托管到协议的注册表智能合约中来进行预付款。租用 NFT 时，智能合约将使用权分配给租用者。

对于利用这一点的项目，他们需要支持我们的无抵押解决方案来加载租用资产。这可以通过 ping 我们的 reNFT SDK 轻松完成。
