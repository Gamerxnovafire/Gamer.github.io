# Official DogeX

狗狗币正在走向一美元！现在，DogeX 已经到来，为狗狗币征服 NFT 空间。DogeX 以执行火星任务的一万个可收藏角色的形式将 Doge 传奇变为现实。NFT 系列将首次展示五组角色，每组都比上一组更稀有：Earth Doges、Moon Doges、Journey Doges、Mars Doges 和 Asteroid Doges。我们是否忘记提及比稀有的十个更稀有的人，包括 Elon Doge、Doge Cuban 和 Billy Doge？

加入我们勇敢的犬类太空漫游者，他们克服了不可能拯救地球的困难，并根据狗狗币的原则找到了一个新的文明。DogeX 向最重要和最著名的狗狗币人物以及他们对狗狗币的抱负致敬。

DogeX 由 Matt Wallace 和他的 Final Stand 社区呈现，以庆祝狗狗币在全球的崛起并每天都做得很好。
