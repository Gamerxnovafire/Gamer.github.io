# by Val Kilmer X Remo

“Love Heals”是 Kilmer 大量剪贴簿中的一部分，这些剪贴簿由广告、名人媒体、艺术书籍和目录以及继续深入研究媒体中名人和偶像制作本质的个人物品组成。

基尔默说，他将自己的自传融入了这些作品中：“出于对被人发现的恐惧，或者只是为了成为原创者而努力，我将我一生的愚蠢融入了这些剪贴簿动作图像和记忆中，以创造一种情感体验，这反映了我，我从我最喜欢的艺术家和作家的作品中得到什么。”

合作者 Remo 是一位跨学科的艺术家，在许多媒介中跨越流派。他的作品融合了概念、流行和街头艺术，使用最新技术创作，通常与生成的人工智能相结合。他的作品曾出现在卢浮宫、莫斯科博物馆、泰特和沙发上。他的街头壁画出现在全球范围内，他创作了 Devo、Donna Summer、Gwen Stefani、Kiss 等作品。
