# round21 - Fizzy NFT Unlock

我们将艺术带到意想不到的运动画布上。不是为了收集灰尘，而是去玩家拿它的地方。无论在哪里，让演奏的人感觉充满活力、真实、灵感和联系——所有时间。我们的。

在我与热爱运动的运动员、企业家、艺术家和艺人共度数十年之后，第 21 轮的想法出现在 2020年。

round21旨在激发我们所有人的创造精神。我们是一群体育迷、运动员、企业家和艺术家，他们相互联系并共同成长——所有时间。我们的。

round21跨越了数字世界和物理世界，并利用技术将我们拉得更近，同时将物理产品和现实世界作为我们的基础。我们称我们的方法为“数字泡沫”，这是我们的秘诀。
