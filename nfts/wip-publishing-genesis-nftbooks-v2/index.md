# WIP Publishing Genesis NFTBooks V2

准备好加入革命了吗？史诗。

我们是一个由作家、书籍爱好者和 NFT 收藏家组成的社区，他们围绕着推动创意写作的使命而聚集。 

我们的第一个概念是 NFTBook Genesis，一个非常基本的平台，类似于我们都非常熟悉的网页，除了它不存在于互联网上，它存在于 IPFS 和它所铸成的区块链上！这为作者提供了高度的控制权，并使他们免于需要第三方参与者在他们不在时将他们的内容保持在线。它还有助于确保托管服务不会使新作家或年轻作家的写作成本过高，因为他们还没有足够的现金流来确保建立一个优秀的网站并保持正常运行。

NFTBook Genesis 对象有 3 个组成部分：铸造到以太坊或 MATIC 区块链的实际 NFT；一个集线器，它很像一个网页，只是它不像 Medium.com 在互联网上那样在互联网上；和内容，包括 PDF 全文等。每个 NFTBook 本质上都是关于特定作者和书籍的可交易平台，作者可以更新此内容以包括文本翻译、购买实体书籍、有声读物的链接等。
