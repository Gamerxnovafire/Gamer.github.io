# You Only Live Once

您也有机会将您的流动性长期投资于我们的dApp，或通过我们的合作伙伴。作为回报，dApp将从$YOLO DAO基金中为您提供流动性奖励。您可以在我们的媒体页面上找到更多关于DAO基金的信息，请随时查看。
*我们只能对一个项目进行风险评级，这不是保证，我们只能披露我们所发现的并向公众提供所有可用信息。
