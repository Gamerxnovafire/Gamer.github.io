# MarsColony

MarsColony 是一个火星殖民框架，它将视频游戏与 NFT 所有权、基于 DeFi 的经济、DAO 治理、行会和跨区块链殖民地相结合。
MarsColony 是 Harmony 上的顶级游戏项目之一。
现在我们正在向 Polygon 链上的新宇宙扩展，并向新玩家开放。
