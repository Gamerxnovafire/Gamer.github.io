# AceStarter

**AceStarter是一个简单的基于层的系统**，为所有层持有人提供所有层持有人的保证分配区块链游戏的ID和IGO。



AceStarter是加密投资者获得最佳策划的世界级区块链项目的全球门户。



全球游戏多链项目的必备入口，也是加密投资者的**最佳ROl启动台**。

**AceStarter**是下一代启动台，在全球范围内策划和启动世界一流的加密项目。
