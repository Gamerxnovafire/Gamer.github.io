# MXC Genesis

MXC - DROP 是 80 位艺术家之间的合作，包括插画家、动作设计师、生成 AI 艺术家、3D 创作者、体素艺术家、涂鸦艺术家和摄影师，MXC - DROP 是共同愿景的表达。这样的收藏旨在展示在各自领域享有盛誉的艺术家以及值得认可的新兴艺术家。两组艺术家将共同吸引那些以低价寻找高价值的价值猎人。随着这一下降，MetaCollective 创造了一种罕见的价值，即所有收入在创作者和艺术家之间平均分享的价值。该集合由静态和动态作品组成，共有 1600 个 NFT。这个系列将是一个盲目的薄荷，它将在一周内揭晓。您获得梦想艺术品的机会与您拥有的薄荷糖数量直接相关。
