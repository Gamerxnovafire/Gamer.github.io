# SuperNormal x Waterbomb League Official

在积极的研究中，观测到了一颗名为“WA-8051B”*的系外行星，并证实这颗行星上有相当数量的水。 对于这个星球的水资源，这可能是人类最后的水和机会，世界范围内的竞争，最终公平竞争，胜者决定占领这片水域。

各国领导人竞相决定自己最擅长的运动，通过领导人的提议，决定选择很久以前因严重水污染而在全球范围内被禁止的运动“水枪大战”作为游戏项目。

终于，在遥远的外星球 WA-8051B，这将是地球最后的希望，建造了水枪战的体育场，每个国家都选择派出球员参加水枪战……
