# Samurices

Samurices 是 7,777 种独特的 Onigiris 的美味集合，由 200 多种特征组合而成，其中一些特征比其他特征更罕见，所有灵感都来自以太坊区块链上的食物和亚洲文化。

所有厨师学徒的开始！ 就像武士道跟随武士一样，您将在与我们一起的整个旅程中被引导去拥抱烹饪士道的神奇之路。 关于我们将如何以及如何共同构建 Samurices 宇宙的有意义的愿景和理念。

作为充满激情的创意者，我们首先将注意力集中在艺术上，而这种精神将定义和塑造 Samurices 的世界。 没有虚假的承诺，只有一个确定性：艺术是并将继续是该项目的基础。
