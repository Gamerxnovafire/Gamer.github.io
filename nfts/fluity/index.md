# Fluity

Fluity Protocol 是币安智能链上 Liquity 协议的一个友好分支。 Fluity 致力于实现 Liquity 的愿景，即提供无息贷款、高资本效率和抗审查稳定币的去中心化借贷协议。
Fluity 具有与 Liquity 相同的功能（开库借入 FLUSD、赎回、稳定池、质押等），只是来自稳定池和流动性池的 FLTY 奖励将发送到归属合约，而不是直接返回给用户。
