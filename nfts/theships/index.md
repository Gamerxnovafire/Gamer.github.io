# The Ships (Expansion)

前往不同的岛屿并与他们进行贸易

文明在它们能够开花之前就崩溃了，这些领域处于动荡和毁灭的不断循环中。有些人找到了持续数年甚至几代人的平静和幸福的口袋，但所有人都遇到了他们的最终命运——出生在看到自己的末日临近而无法改变的世界中，他们灭亡了。仿佛这一切都是命中注定的，是一场宇宙游戏的一部分，规则如此神秘，甚至先知都无法理解或预测它们。他们也灭亡了。

建造者忽略了一些细节，一些如此重要的东西，以至于建造者思想的错综复杂的网络无论以何种方式显现为物理现实，它似乎都不正确。有什么东西不见了。这不是建造者的梦想。
