# Real Apes Club Nft Official

Ascended 是 8,888 名随机生成的独特探险家的集合，他们寻求面具的奇异神秘力量。加入他们，冒险进入 Inmanis 的超现实和超自然世界，寻找他们！

Inmanis 的世界广阔而贫瘠。

总人口未知，但被认为是稀疏的。它的居民大量聚集在称为绿洲的地方。这些避风港彼此隔开无边无际的距离。

事实上，这些绿洲之间的世界一直被单独留下，因此被笼罩在默默无闻中。众所周知，无畏和鲁莽的人会出发，试图策划和驯服这些荒野。这些探险从数周到数年不等，有些……有些还没有返回。

奇怪的是，在比较了各种探险的记录之后，这些无边无际的区域似乎并不是一成不变的。每个探险家的故事都与上一个不同。Inmanis 似乎是不屈不挠、千变万化和不屈不挠的。
