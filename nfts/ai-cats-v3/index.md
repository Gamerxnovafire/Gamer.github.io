# AiCats

作为提供对区块链数据的公平访问的一种方式，我们开发了**Etherscan 开发人员 API ，使开发人员能够通过****GET/POST 请求直接访问 Etherscan 的区块浏览器数据和服务。

##### ▶ 什么是 AiCats？

AiCats 是一个 NFT（非同质代币）集合。需要通过存储在区块链上的数字艺术品集合。
