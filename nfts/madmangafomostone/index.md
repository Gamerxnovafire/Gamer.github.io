# MadManga FOMO Stone

MadManga 是首个由社区驱动的收藏品项目，旨在为漫画爱好者和创作者在 web3 世界中创建集英社。目前，MadManga 在以太坊区块链上总共提供了 4,200 个精选的 FOMO Stones，FOMO Stone 持有者可以访问 MadManga 世界。在 MadManga，我们相信艺术家、漫画爱好者和 web3 爱好者将团结起来，共同建设一个更好的去中心化未来。

我们的使命是在区块链上建立最大的漫画社区和最好的漫画故事。在 Mad Manga，我们的目标是为充满激情的创作者、漫画家、故事讲述者和粉丝提供最全面的策展平台，以与众多独特功能并存的大型漫画系列相媲美。
