# LEP721

2017 年，在以太坊区块链上免费铸造的 CryptoPunks，今天我们再做一次。为了纪念 CP 生日，我想给整个 NFT 社区和整个空间的艺术收藏家送上礼物，让他们能够再次感受这种情感LEP (LowEffortPunks) 创作者的我想向这两位创作者致敬，并免费提供给所有人。没有路线图。二级创作者只有 ART、FREE MINT 1xWallet 和 0（零）版税。CC0 项目感觉免费。我们是自由的，我们将永远是自由的。不隶属于 LarvaLabs (YugaLabs) 或 LowEffortPunks。

LEP721 是 2000x2000 手绘艺术图像，通过算法生成。大多数都是看起来很朋克的男孩和女孩，但也有一些更稀有的类型：猿、僵尸，甚至是奇怪的外星人。每个 LEP721 都有自己的个人资料页面，显示他们的属性以及他们的所有权/待售状态。
