# Bend Ape Yacht Club

Bored Ape 游艇俱乐部是 10,000 个独特的 Bored Ape NFT 的集合——生活在以太坊区块链上的独特数字收藏品。您的 Bored Ape 可兼作您的 Yacht Club 会员卡，并授予会员专属福利，其中第一个是使用协作涂鸦板 THE BATHROOM。社区可以通过激活路线图来解锁未来的领域和福利。

每个 Bored Ape 都是独一无二的，并以编程方式从 170 多种可能的特征中生成，包括表情、头饰、服装等。所有类人猿都是毒品，但有些比其他人更罕见。

猿作为 ERC-721 令牌存储在以太坊区块链上并托管在 IPFS 上。购买一只猿需要0.08 ETH。

要访问仅限会员的区域，例如 Apeholders 需要登录他们的 Metamask 钱包。
