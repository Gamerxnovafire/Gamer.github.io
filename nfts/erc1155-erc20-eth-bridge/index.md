# ERC1155<->ERC20<->ETH bridge

以两种方式（锁定或回绕）在 ETH、ERC-1155 和 ERC-20 之间进行双向转换。所以 2x2+1=5 种转换方式。不收费。

将任何 ERC1155 包装在 ERC20 中，反之亦然
使用高级智能钱包（赚取、投资、找回丢失的账户、设置每日转账限额、使用人类可读的地址、无交易费用、简单的 DeFi 等）而不是普通的以太坊账户
