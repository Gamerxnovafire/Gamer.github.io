# Sensi Legends

###  森西传奇史诗

足球历史充满了令人难忘的时刻，这些时刻是我们集体记忆的一部分。

Sensi Legends EPICS 是手动创建的一种 NFT 的特别版，用于庆祝足球历史上的特殊时刻，例如：

- 马诺·德·迪奥斯·马拉多纳。
- 帕内卡点球。
- 克里斯蒂亚诺·罗纳尔多自行车踢
- 伊吉塔的蝎子踢
- 埃德的 2016 年欧洲杯目标
- 齐达内对马可·马特拉齐的头撞Sensi Legends 是一个 NFT 系列，灵感来源于经典的 Sensible Soccer。每个 NFT 都是以编程方式生成的，包括根据比赛、团队和玩家随机和手动选择的属性，使每个 NFT 成为独特的数字收藏品。
