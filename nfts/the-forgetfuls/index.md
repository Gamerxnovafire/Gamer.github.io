# The Forgetfuls

欢迎来到 Forgetville... 一个被遗忘的乡村小镇，被 6,543 名当地 Forgetfuls 占据，他们在以太坊区块链中漫游。欢迎来到忘恩维尔。
一个被遗忘的乡村小镇，被当地的健忘者占领。早在1827年，镇上就爆发了一场来自当地工厂的危险气体，对人类的思想造成了严重的副作用，消除了使用“常识”的能力，这使得最简单的日常活动看起来非常困难。

居住在Forgetville镇的其余人员是错过了最后一班疏散列车的家庭和个人，看来他们已经受到危险气体的影响，最终他们“忘记”赶上火车。现在，他们以自己的方式在城镇周围快乐地生活。

作为一个团队，我们访问了Forgetville，采访了一些当地人，他们非常善良，他们张开双臂欢迎我们 - 我们认为。

我们确实意识到有一个非常有趣的气味云，漂浮在城镇周围，正在慢慢接近我们。等等，这个故事要去哪里了...
