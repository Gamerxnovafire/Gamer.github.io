# LAYC

Lazy Ape Yacht Club 是一个高端混合集合，由 10,000 个独特的 NFT 组成，生活在以太坊区块链上。 您的 Lazy Ape 将授予您专供会员使用的未来福利。 Lazy Ape Yacht Club 是 Apeverse 的“扩展包”。 LAYC 持有者将通过我们的合作伙伴以及与 Ekta Chain、Rich Fox Club、Apes Together Strong 等的合作获得独家实用程序。

LAYC 是一个慈善机构，支持爱丁堡和洛锡安的社区青年和儿童工作。 我们支持一个由 120 多个青年和儿童组织组成的网络，近 3000 名员工和志愿者参与了超过 28,000 名儿童和青年的活动。
