# TotemFi

图腾Fi是什么？
不仅仅是一个预测市场。 TotemFi 是一种将赌注游戏化为社区的协议。 我们一直在努力打造一个有趣、简单且非惩罚性的平台。

不仅如此……我们的平台将允许用户参与预测市场，而不会因为不正确而受到惩罚。 对于加密初学者来说，这允许在没有风险的情况下进行接触和教育。 经验丰富的用户可以从质押中获得有保证的回报，以及正确预测的 TOTM 和 BTC 奖励； 这具有出色的投资组合多元化优势。
