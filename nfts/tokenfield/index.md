# TokenField

什么是 TokenField？
TokenField：赚取加密货币的最聪明方法

TokenField Token，以下简称 TF，是一种符合 ERC-20 标准的智能合约，专为部署在以太坊区块链上而设计。 TF 是一种去中心化、公平发行、自动流动、无需信任可交换、有息、类似债券的代币。

TF 通过以正确的方式进行加密来重新定义 DeFi：零团队代币、90% 或更多的预售资金用于 Uniswap 流动性、Contract BURNS Uniswap LP 代币、无管理员密钥。 如果您正在寻找赚取加密货币的最佳方式，那么您来对地方了。
