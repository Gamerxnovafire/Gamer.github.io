# Loot Resources

冒险家资源[资源是用于制作战利品]的随机原材料。

发现 Loot 的原始人是有福的。为了在这个世界上生存，需要许多冒险者和他们的战利品。

虽然不知道长剑、恶魔皮腰带和其他原始战利品是如何产生的，但怀疑它们是由最近发现的资源制成的，例如钢铁、恶魔皮或皮革。

这些资源究竟是如何被制作成新的战利品仍然是一个谜，是否可以创造出全新的未被发现的物体也是一个谜。

一些研究人员推测，将资源制作成战利品可能需要某种形式的能量、魔法或冒险试验。其他人则认为，尽管可能会制作新的战利品，但原始战利品可能会充满无法制造的活力。所有人都注意到，奇怪的是，资源包从未包含[黄金](https://twitter.com/WillPapper/status/1433243941449568259)，因为它可以在其他地方找到。

只有研究和实验才能揭示什么是可能的。
