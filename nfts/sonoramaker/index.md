# Sonora Maker

Sonora Maker 是基于 Remilia Collective 的 Milady Maker 的衍生项目。作为一个艺术项目，Sonora Maker 对源材料的参与没有任何讽刺意味，也无意模仿或讽刺。Sonora Maker 旨在作为对 Milady Maker 的评论，展示后作者身份如何促成去中心化的模因世界构建。该项目旨在举例说明放弃所有权如何导致一个想法的指数增长和扩散，特别是关于米拉迪作为一个过分人物的性质。
Milady Maker 衍生作品： Milady Sonora 的原创艺术，由 Modesty Preserver 改编。
正面视角：索诺拉斯解除武装的表情打破了第四面墙，直接向观众讲话。
免费铸币厂： Sonora Maker 是 Milady Maker 持有者可用的 444 个 NFT 的免费铸币厂。
