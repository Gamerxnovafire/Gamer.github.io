# Meta Adventure Kongz Collaboration

Kongz 宠物是动画角色，将跟随元冒险家在元宇宙中的旅程。确保他们得到他们需要的所有爱！

Meta Adventure 是基于以太坊区块链的 NFT 集合。 NFT 的所有权将使您能够访问您在我们的元宇宙中的化身，随着时间的推移，它将不断地托管更多的游戏和其他内容。您的头像将能够玩我们的 P2E 游戏，并通过这样做获得我们的游戏内货币 Meta Gaming Tokens (MGT)。这种货币将有很多用途，包括升级你的 NFT 统计数据和特征，但也让他们为他即将面临的考验提供最合适的装备！
大多数 P2E 游戏正是它们所代表的：你玩游戏只是因为你想从中赚取一些东西。我们的愿景是发布令人愉悦的游戏玩法和高品质的美术作品，以提供出色的游戏体验，同时获得奖励，让您争夺排行榜并获得独特的奖励，或者只是享受看到您的角色不断成长的过程。
