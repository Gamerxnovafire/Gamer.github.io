# Decentraland

Decentraland 市场允许用户浏览、购买、出售和管理 LAND 代币。 Marketplace 是一个建立在以太坊上的完全去中心化、零费用的应用程序。

测试你的想象力极限
使用简单的 Builder 工具创建场景、艺术品、挑战等，然后参加活动赢取奖品。对于更有经验的创作者，SDK 提供了让世界充满社交游戏和应用程序的工具。
