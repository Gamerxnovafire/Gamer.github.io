# Kidrobot Zodiac

现在铸造Still Collectibles：ETH 上的 475 版动画收藏品：ETHDUNNY 上的 95 版是由 Kidrobot 和 Tristan Eaton 创作的具有收藏价值的设计师艺术玩具。 Kidrobot 发布了收藏限量版 Dunny 设计师艺术玩具和 Dunny 系列，其中包含来自世界上最具开创性的艺术家、时装设计师、插画家和标志性艺术传奇人物的设计。 在过去的二十年里，Dunny 已经成为一个标志性的艺术平台，通过 Kidrobot.com 和全球精选零售商提供的限量版作品，将新兴艺术家的艺术作品带入公众视线。该系列由 Kidrobot 正式授权。
