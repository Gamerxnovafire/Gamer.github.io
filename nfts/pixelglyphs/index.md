# Pixelglyphs

什么是像素字形？
Pixelglyphs 是一组 10,000 个独特的链上化身 NFT，使用以太坊区块链上的细胞自动机创建。

您的 Pixelglyph 是在铸造过程中随机创建的。元胞自动机算法在链上运行，首先是 NFT。像素数据和颜色存储在链上。您可以随时使用代码重新创建您的 Pixelglyph。

您的 Pixelglyph 可以在互联网上和您最喜欢的 dApp 中充当您的匿名头像。

您拥有的每个 Pixelglyph 都将允许您兑换一个“pxg.eth”域 NFT。

Pixelglyph 所有者将能够兑换 pxg.eth 域名。我们扩展了 ENS 名称以创建一个专注于 NFT 的配置文件，该配置文件利用 ENS，但增加了增强的 NFT 功能。
