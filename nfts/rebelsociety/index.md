# Rebel Society

Rebel Society 是一个 NFT 限量版系列，包含 7000 个女性角色，由手绘元素创作，以庆祝所有鼓舞人心的坏女人，并作为 ERC-721 令牌存储在以太坊区块链上。

通过拥有 Rebel Society NFT，您将获得独家会员特权，例如赠品、投票权、未来的投递和合作。
通过 Rebel Society 倡议，我们希望帮助其他女性艺术家通过区块链路径将她们的艺术带到世界各地。

从首个系列的销售中收集的部分资金将用于赞助即将推出的新女性艺术家，这些女性艺术家凭借自己的才华脱颖而出，但由于所涉及的成本而无力推出系列。
‍艺术家将在 Rebel Society NFT 持有者和 Rebel Society 团队的帮助下进行预选。
