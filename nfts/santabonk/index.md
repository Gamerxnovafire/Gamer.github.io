# SantaBonk

SantaBonk 是 BSC 上一个社区驱动的代币，旨在回馈。每个持有 SantaBonk 的人都在不错的名单上，因为 Santa Bonks 每小时以所有交易价值的 11% 向持有者发放 BUSD 奖励。您需要做的就是持有 SantaBonk，您的钱包就会收到源源不断的被动收入。
