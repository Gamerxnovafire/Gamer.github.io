# Pudgy Penguins


**什么是 Pudgy Penguins矮胖企鹅？**

矮胖企鹅是“不可替代的代币”以太坊区块链，以及每个 PUDGY PENGUIN 旨在最大限度地营造氛围。

LIL PUDGYS 是 22,222 名北方血统的矮胖子。LIL PUDGYS 很可爱很可爱，但不要让他们的外表欺骗了你。他们凶猛、强大，并且是对抗负面邪恶熊的盟友。

这只矮胖的企鹅是在冰冻的以太坊区块链上四处滑动的 8,888 只企鹅之一。企鹅是在冰冷的北极地区发现的矮胖企鹅社区的正式成员。

欢迎来到矮胖企鹅。我们是我们的企鹅，我们的企鹅就是我们。
