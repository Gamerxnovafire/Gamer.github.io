# Meta Girlfriends

Meta Girlfriends 即将来到 Metaverse。 每一个都是随机生成的，并使用 20 个类别的 600 多个特征精心制作。 每个 Meta Girlfriend 都可以使用 Rainbow Room 进行定制。项目中的每个 NFT 都有三个视图：Public PFP - 公众只能看到裁剪后的 PFPMembers Only - 可以在我们的网站上看到每个 NFT 的全身艺术品私人视图 - 持有者可以 脱掉他们的 Meta 女朋友的衣服，看到每个人都完全裸体彩虹室允许您组合两个 Meta 女朋友并选择要保留的特征！ 持有者可以混合和匹配特征以自定义他们的 Meta Girlfriend 或创建主题。 这使得收藏变得通货紧缩，游戏化变得有趣并且越来越有价值。通货紧缩 - 请记住，该项目是通货紧缩的。 随着 Meta Girlfriends 的合并，其中一个 NFT 被烧毁。 随着持有者烧掉他们的 NFT 以打造完美的 Meta Girlfriend，总供应量将继续减少。
