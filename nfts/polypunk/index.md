# Polypunk

这 10,000 个 PolyPunks 中的每一个都具有根据定义的稀有系统使其独一无二的属性。

Polypunks 是一个生活在 Polygon 链上的 NFT 项目，于 2020 年 7 月 3 日推出。虽然不同的链上有不同的朋克，但 Polygon 缺乏自己的朋克，因此 Polypunks 被创建为 Cryptopunks 的 1:1 副本。Polypunks 认识到，随着 Polygon 的低交易费用、不断增长的用户和越来越受欢迎的趋势，基于 Polygon 的代币创建 NFT 是必要的。他们的目标是提高 NFT 在 Polygon 上的可见性并鼓励新兴项目，同时保持其在该类别中领先项目的地位。Polypunks 希望很快在更多的 NFT 市场上上市，以增加曝光率。Polypunks 在 Opensea.io 和 Lootex.io 上列出。
