# The 8102: Yachts

格里是谁？
Gerry 是一个古怪的白手起家的亿万富翁，他喜欢一头扎进疯狂的项目中，通常只有他的私人助理 NG（“Numbers Guy”的缩写）才能从亏损中拯救出来。起初，当 BAYC 公开他的会员照片时，他非常生气，因为 Gerry 没有阅读细则，实际上是因为事先喝了太多香蕉牛奶而有点陶醉（他在选择服装时还混淆了“无聊”和“傻眼”和表情）。但他决定顺其自然，因为没有人会因为太阳镜而真正认出他。

由于格里的爱好之一是探索兔子洞，他可能会也可能不会接受阴谋论和各种边缘话题，例如时间旅行、不朽、超级人工智能、外星人和太空探索。他先说话后思考的倾向在过去造成了一些“问题”，因此，他公司的董事会（又名“老他妈的”）一致投票支持删除 Gerry 的社交媒体资料。只有经过 PR 培训的 NG 才能在社交媒体上发布有关 Gerry 的内容，但仍然有一些持续的谣言称 Gerry 是加密社区中一些备受瞩目的 Twitter 账户的幕后推手。
