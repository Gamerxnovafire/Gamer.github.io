# Rare Liquid Society

我们很高兴地宣布，Rare Liquid Society 成员的第一次空投已经准备就绪。

Cloud Cellars 是个人 web3 域，用于共享、展示、存储、审查和交易 Rare Liquid 的资产支持的 NFT。

由 Rare Liquid 存储和保护的实物葡萄酒和烈酒收藏的可共享数字展示。

Cloud Cellar 是一个实时应用程序，允许所有者登录并自定义 url 和外观。外观源自 Cloud Cellar NFT 调色板。

任何用户都可以在您的 Cloud Cellar 中点赞、评论或竞标 NFT。Cloud Cellar 是参与 Rare Liquid 的实体酒和烈酒掉落所必需的。

随着时间的推移，Cloud Cellars 将获得治理 RLIQ 代币的一部分。
