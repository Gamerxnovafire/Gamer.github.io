# Purple Ape Club

Purple Ape Club 是一款令人愉快的元界 NFT 游戏，包含 5,555 个在多边形区块链上随机生成的 

NFT，由 150 多个独特的手绘特征组合而成。



##### ▶ 什么是紫猿俱乐部？

Purple Ape Club 是一个 NFT（Non-fungible token）集合。存储在区块链上的数字艺术品集合。



##### ▶ 紫猿俱乐部代币有多少？

总共有 5,555 个 Purple Ape Club NFT。目前，480 位车主的钱包里至少有一个 Purple Ape Club NTF。



▶ 什么是流行的紫猿俱乐部替代品？

许多拥有 Purple Ape Club NFT 的用户还拥有 PicaLions、 Manga Ape Universe Club、 Bored 

Amazing Ape Club和 Save the Martians。
