# MettiForRoeVWade

Metti For Roe v. Wade 是 Pumpametti 的 NFT 艺术收藏品，旨在支持女性的选择权。SCOTUS 对 Roe v. Wade 的推翻进一步使美国远离了美国宪政民主的核心原则之一，即政教分离。免费铸造 1000 件艺术品，其中 300 件留给 OG Pumpametti 收藏家。该系列的所有版税将用于支持为女性生殖权利而战的非营利组织。最终，Pumpametti 使用了所有历史和现代的主题和特征，创造了一个未经过滤、幽默和开放的艺术空间。他捕捉到艺术的原始能量，不受任何规则或规定的束缚，让观众坐下来感受他们的原始反应，无论是积极的还是消极的。就像 Austin Lee 作品中那些色彩斑斓、与众不同的人物一样，Pumpametti 还创造了一种可识别且独特的风格，同时创造了一系列涉及其他目的和主题的不同作品。很难说Art Brut和外来艺术的实践未来会走向何方。
