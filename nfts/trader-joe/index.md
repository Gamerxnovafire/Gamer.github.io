# Trader Joe

什么是交易员乔？
Trader Joe 是 Avalanche 区块链原生的一站式去中心化交易平台。 Trader Joe 快速、安全地构建，旨在为 DeFi 前沿的社区服务。 在一个完全集成的屋顶下最大化您的收益生成活动，为您提供独特、创新和高度可访问的 DeFi 体验。

分享掉期产生的平台收益。 掉期费用产生的所有协议收入 (0.30%) 分配给流动性提供者 (0.25%) 和 sJOE 质押者 (0.05%)。

用户喜欢我们，因为我们通过有趣和引人入胜的功能提供 Avalanche 上最好的用户体验。 除了提供出色的用户体验外，我们还快速构建新功能并提供最佳交易价格和最广泛的交易代币选择。
