# RichApesClub

当您购买 Rich Ape 时，您购买的不仅仅是头像、JPEG 或稀有艺术品。您正在获得一个专属俱乐部的会员资格，该俱乐部的福利和产品将随着时间的推移而增加。您的 Rich Ape 可以作为您的数字身份，并为您打开数字领域的大门。                     
每个 Rich Ape 都授予其所有者访问会员独有的特权，包括访问会员独有的不和谐频道、商品、虚拟和现实生活体验。我们来自世界各地的 Nft 投资者和收藏家社区就是公用事业公司。
我们将为所有持有者举办为期一年（52 周）的每周直播加密教育课程。内容将按需提供给所有持有者。
