# C86 Cyborg

作为官方 DAO，Cyborg 86 是 8,600 个独特的 Cyborg NFT 的集合，它们存在于以太坊区块链上。 我们在我们的企业家 Metaverse 俱乐部里放松。 在这里，您可以与我们对 Web 3.0 感兴趣的企业家社区发展您的网络

CYBORG 86代表了8，600个独特的半机械人的集合，这些半机械人按稀有度分类，并生成了数百种元素。
