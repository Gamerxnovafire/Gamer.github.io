# AnRKey X

Oracle Shard NFT 是在 Polygon（第 2 层以太坊）上铸造的游戏卡，可在整个 AnRKey X 宇宙中提供游戏实用程序。这些 NFT 可以通过在 Battle Wave 2323 等游戏中质押来免费提高 APY（大幅增加您的团队奖励），并在 NFT 市场上收集或出售以获取利润。

Battle Wave 2323 是 AnRKey X 发布的第一个 Play to Earn 游戏 dapp，为游戏玩家和内容创作者提供了通过 DeFi 和 NFT 获得奖励的有趣新机会。用户通过收集、质押和玩 APY Boosted NFT、构建 NFT 游戏卡组以及玩令人兴奋的迷你游戏来竞争赢得每周的奖池。
