# SMALL WRLD Genesis

SMALL WRLD 是一家科技创业公司，作为 NFT 项目推出，专注于通过可扩展的 Web3 解决方案推进旅游业。在产品方面，我们从一家基于区块链的旅行社开始，并朝着扩展为著名 Web3 旅游软件提供商的愿景迈进。

我们的故事始于 Genesis NFT 系列，授予其持有者私人协商的旅行优惠和福利、独家旅行社优惠、紧密联系的旅行爱好者社区和未来的旅行应用程序。


Web3 技术具有积极颠覆旅游业的巨大潜力。SMALL WRLD 的目的是站在旅游公司的最前沿，通过 NFT 和代币化等区块链解决方案重新定义行业，以构建更加用户驱动、灵活和可靠的数字旅游产品。借助 Web3 和 NFT 的力量，我们可以创造消费者和公司双赢的旅游公司未来
