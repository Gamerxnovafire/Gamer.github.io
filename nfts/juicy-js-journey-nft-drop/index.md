# Juicy J's Journey NFT Drop

Juicy J的旅程

Juicy J 和 ASTERISK* 与 Everipedia (IQ) 合作推出了 Juicy J's Journey，这是一个限量的 NFT 集合，以庆祝 Juicy J 的最新专辑 The Hustle Still Continues 的推出。该系列使用区块链技术、数字艺术品、Juicy 音乐视频中的场景、元数据和乐器来讲述 Juicy J 作为嘻哈偶像、加密货币支持者和大麻企业家的故事。这些作品还纪念 Eternal Impact，这是 ASTERISK* 的最新产品，这是由 Juicy J 和 Gary Vee 通过他的大麻行业营销机构 Green Street 创立的创新大麻品牌。

Juicy J、ASTERISK* 和 Everipedia 联手打造了 Juicy J 的旅程，这是一个包含 5 件 NFT 艺术品的集合，每件作品限量 84 个版本，总共仅 420 件。所有 5 幅 NFT 艺术品都包含由 Juicy J 和粉丝使用他的鼓和循环套件共同制作的节拍。该集合还包含对加密货币和 IQ 令牌的引用。Juicy J 是区块链技术的早期支持者。早在 2019 年 2 月进入加密货币寒冬时，他就看到了区块链技术的潜力，并邀请 Everipedia 联合创始人兼总裁 Sam Kazemian 在他的“让我看看”音乐视频中加入 IQ 代币的代码。该代码被合并到“让我看看那个代码”NFT 中，这是构成该系列的五件 NFT 艺术品之一。Juicy 总是忙于从说唱到大麻再到加密货币 The Hustle Still Continues。
