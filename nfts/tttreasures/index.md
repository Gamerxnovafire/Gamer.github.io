# TTTreasures

元宇宙中的童年小吃
TTTreasures 是一个以宝石饼干为设计核心的 NFT 系列，体现了每个持有者的重要性。

宝石饼干作为新加坡传统小吃带来的怀旧之情，我们相信它也会给持有者灌输一种归属感。

信任、凝聚力、健康！

大约 80% 的销售收入将进入称为 TTTreasures 的社区金库。

我们的目标是聚集志同道合的资本家、研究人员和建设者，他们在 P2E / Metaverse 领域拥有无与伦比的专业知识，并拥有一个由数字资产和加密资产组成的集体投资组合。

TTTreasures 基金的多元化使我们能够最大限度地降低风险并稳步增长。 提案将由 TTTreasures 持有者投票决定。
