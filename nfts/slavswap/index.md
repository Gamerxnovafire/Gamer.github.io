# Slavswap

Slavswap Finance 是一个运行在币安智能链和 Pancake 交易所上的去中心化交易所，还有许多其他功能可以让你赚取代币。
我们正在尝试做的是创建一个永久通缩代币伏特加，它允许具有足够燃烧机制的恒定价格暴涨。我们不是试图取代掉期和交换，而是为系统增加价值，并为人们创造一个合适和可持续的环境来生产高年利率的农场。交换和蹲在币安智能链上最醉的 AMM 和 YFP。
