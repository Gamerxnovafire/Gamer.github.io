# YOLOrekt

下一代期权交易，游戏化。YOLOrekt是一个交易者对交易者的dApp，交易者在ΞEth / $USD的最终价格上下注，在快节奏中，三分钟的游戏分为三个阶段。通过融合直观的设计，游戏化和有趣的社交氛围，YOLOrekt为短期期权交易提供了新的旋转，新老交易者都将玩得很开心。YOLOrekt是一种引人入胜，有趣，社交且可能有利可图的方式来竞标加密货币的未来价格。|直观的|超游戏化|社交 fi |该平台的核心使能技术提供了在数秒内在区块链上结算市场的能力。作为我们的第一个产品，用户将能够通过对结果进行出价来预测资产的最终价格是高于还是低于执行价。这是“出价上升”或“出价下降”的游戏，允许交易者只需几支蜡烛就可以得到结果。通过直观的用户界面，可以在任何可用的市场中进行出价，从而使期权交易尽可能容易。
YOLOrekt将允许用户预测各种流行资产的短期波动，包括比特币（BTC）、以太坊（ETH）、特斯拉（TSLA）、GameStop（GME）等。作为第一个产品，预测游戏基于3分钟的运行时间，每次结算后重复。用户只需出价，然后选择价格在3分钟内是上涨还是下跌。
用户将能够预测最终价格是高于还是低于执行价，并对结果进行投标。该平台的核心是能够在几分钟内启动和结束游戏。这些都是“出价上升”或“出价下降”的游戏，让交易者只需几支蜡烛就能赢/输。它为投标双方提供了平等的胜负机会，使每个人的期权交易更加容易。
网络产生的费用将在YOLO代币持有人和流动性提供商之间分配，并向财政部分配少量费用。
