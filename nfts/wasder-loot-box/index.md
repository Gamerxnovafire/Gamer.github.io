# Wasder Loot Box

什么是重要的
使用 Spaces，您可以获得您喜爱的游戏的所有内容。都在一个地方。

精彩的游戏内容就是生活，对吧？但找到真正重要的东西并不总是那么容易。当然，有些人似乎确切地知道在哪里看。但我们认为这是规则的例外。我们相信每个人都应该拥有一个您可以控制重要事项的地方。那个地方就是 Spaces，您可以根据自己的喜好进行定制。Spaces 围绕您喜爱的游戏收集了 Wasders 的主要功能。通过关注空间，您可以在一个地方发现所有重要信息——从游戏特定内容、社区和活动（即将推出）到派对聊天。这只是开始，所以请注意这个空间。
