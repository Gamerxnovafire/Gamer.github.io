# Prism Finance


**什么是 Prism Finance棱镜金融？**

Prism Finance是一个创新且可靠的双链自动做市商（AMM）去中心化交易所和收益优化器，托管在币安智能链和 Polygon 上。

这个项目背后的想法是为保守的农民创造一个安全的地方。我们的农场和矿池将设置 BSC 和 Polygon (Matic) 网络上可用的不同加密资产（稳定币、BNB、MATIC 等）。

在棱镜金融，我们的专家可以帮助您探索市场上的各种贷款。我们将尽一切努力为您提供最适合您的业务。投资贷款 我们理解购买投资物业是建立财富的好方法。我们在这里帮助您根据您的需求探索贷款方案。
