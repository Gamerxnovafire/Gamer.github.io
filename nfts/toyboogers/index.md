# Toy Boogers

Toy Boogers 系列是 100% 手工制作的，不是以编程方式生成的。所有 3,333 个 NFT 都是由 Toy Boogers 的创造者和创始人 Doug 创作的 1/1 原创艺术品。

背景故事：那一年是 2021 年，一群 NFT degens 开始开发特殊的隐形传输技术，将自己物理地传送到元宇宙中。开发过程非常兴奋，不知何故，鼻屎卡在了机器的聚焦晶体上。他们几乎不知道，这会导致严重的（但令人愉快的）故障。当他们使用这台机器时，当他们传送到虚拟世界时，他们会与收藏中的所有疯狂化身进行组合和重新排列。他们成为了一个史诗般的 NFT 收藏，他们自己称为 Toy Boogers。

Toy Boogers 系列由 3,333 个独特的 NFT 组成，这些 NFT 存在于以太坊区块链上，存储为 ERC-721 代币，并托管在 IPFS 上。
