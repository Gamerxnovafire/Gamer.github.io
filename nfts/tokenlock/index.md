# TokenLock

什么是令牌锁？
将您的流动性——代币和资金——锁定在您自己的储物柜智能合约中。

一分钟锁定智能合约，用于团队代币归属、LP代币锁定、流动性锁定。

所有代币都被锁定在我们经过审核的智能合约中，您只能在锁定时间到期后提取

一个透明、多链、安全和去中心化的智能合约保险库，用于按时释放流动性和项目代币。

多链 dapp，任何人都可以在几分钟内轻松铸币，无需编码经验，可定制且经过全面审核的代币。
