# Significant Others

**Signiscapes - 简介**

[Signiscapes]是“[重要他人]”NFT 系列的下一个作品。有了[Peace]的艺术和我的合同 ( [NftDoyler])，我们希望在原始系列的基础上再推出另一个有意义的产品。

横幅+PFP！为什么不？

虽然大多数系列似乎是横幅*或*个人资料图片，但我们希望将它们组合成一个有凝聚力且独特的系列。

例如，这是一个使用我的 PFP 和用户名的早期演示。

您可能会问的第一个问题是，“但是如果我不想要一个标有‘doyler’的横幅怎么办？”

这些横幅中的每一个的定制实际上都是基于每个用户的！

虽然定制将永久“存在”在链上，但实际上只要用户转移 NFT，它们就会被删除。这意味着无论何时您购买 Signiscape，包括二次销售，您都将从一个全新的定制开始！

希望这可以为集合提供一些想法，以允许在个人基础上进行自定义，而不是将它们直接绑定到资产本身。

注意：这确实意味着如果您将 NFT 买回来，您将需要重新定制它，但如果有足够多的人关心，我们可以考虑改变这一点。
