# Lucky Unicorn Metaverse

Lucky Unicorn Metaverse 是一个运行在 Lucky Network 上的 NFT 游戏平台。允许玩家孵化、饲养和与他们的独角兽战斗。通过公会系统，玩家可以建立自己的帝国，拥有成百上千的独角兽。
