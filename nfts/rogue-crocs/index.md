# Rogue Crocs

历史
Rogue Croc 于 2021 年底在 Fantom Opera 网络上推出，发布了 10,000 个原始 Rogue Croc，目前由 1000 多个独特的持有者持有。 紧随其后的是我们最稀有的 NFT 收藏，Animated Rogue Croc 实用代币（500 个代币）和以太坊网络上的免费掉落（5,555 个代币）
拟议的游戏规则

- 基础知识：玩家将能够放置他们希望下注的 NFT（仅限 FTM NFTS），让可能的对手看到。 对方玩家将放置他们希望与之对抗的 NFT，如果双方玩家都对比赛感到满意并接受，则比赛存在。
