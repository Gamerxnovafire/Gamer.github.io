# Onion Meta Official

OnionMeta 由台湾 Youtuber Onion Man 推出。它是由洋葱侠中的洋葱、阿文、洪杰克、博尔君和婷婷等角色随机生成的 1,024 个 NFT 的集合。OnionMeta 持有者可以参与 Discord 上的独家社区，并有机会成为 Youtube 上的 Onion Man 角色。在不久的将来，所有成员都将访问由 OnionMeta 团队构建的元宇宙，称为“Onionverse”。

是洋葱所发行的NFT，拥有OnionMeta，你就是洋葱宇宙的元祖队员，能有机会参与洋葱实体世界的发展，还能成为虚拟世界的伙伴。
