# Shmokies

Shmokies是我们团队的第一部分，“Magic Carpet Labs”，我们的一小群加密书和老朋友。

Shmokies是以太坊区块链上的CC0（知识共享许可）NFT集合，将数百种独特设计的多色烟雾图案与相应的特征和背景相结合，形成2，500件独特的艺术品 - 当然灵感来自大师本人，Snofro的传奇Chromie Squiggles。我们不值得！

Shmokies作为我们引以为豪的艺术而独立存在，但它们也是我们团队推出的任何和所有未来项目的通行证。即将推出 - 敬请期待！

我们已经定价了Shmokies（前500个免费，最后2，000个0.01 ETH），以便所有人都可以访问，或者正如我们内部所说的“社区发布”。我们希望我们的朋友和任何有兴趣的人都有一个Shmokie。

如果你喜欢他们，薄荷，加入我们的不和谐，我们喜欢反馈和良好的能量。谢谢你，然后醒来！
