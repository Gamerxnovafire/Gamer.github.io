# PridePass

PridePass 是第一个结合 IRL 活动、赠品、职业支持和未来收入分享的 Alpha 通行证。令人兴奋的是，大部分版税都用于推动 LGBTQI+ 倡议和支持的组织。这些实际上是写入智能合约的！

为什么不？那里有许多令人惊叹的社区，在扩大他们的声音方面值得帮助。PridePass 将是朝着正确方向迈出的又一步。通行证的所有者将获得比我们的造币厂成本更多的好处，这对大多数人来说都是免费的（如果你早点）。

有几种方法可以提供帮助：在 Twitter 上关注@pride_pass
写一点爱和支持的推文，因为我们希望业主喜欢它，让其他人知道。
将我们与具有相似价值观的社区联系起来，这样我们就可以共同建设和帮助
