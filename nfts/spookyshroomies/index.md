# SpookyShroomies

188个独特的次元生物，穿越元界，穿越最佳V.I.B.E。 你来冒险吗？

Shroomies 是一个充满激情的项目，由三个朋友创建，灵感来自奇怪的想法和对这个陌生世界的热爱。任何发现自己拿着 Spooky Shroomie 的人都会在他们的生活中邀请一些令人愉快和奇怪的东西......

到目前为止，可以肯定的是，Shroomies 的持有者会在他们的生活中发现“有趣的事件”。 

因此，请选择与您一起 VIBE 的 Spooky Shroomie - 它是为您制作的。 

此外，作为持有者，您将成为我们享受生活的怪人社区的一员，了解 Shroomies 为何在这里 - 以及他们的动机是什么......

我们将分享诙谐的玩笑、存在的危机、竞赛、食谱和我们不寻常经历的故事。 

我们即将推出一款实用代币，可供持有者免费铸造、现场艺术体验、整合到 Metaverse 的重点以及更多计划……但我们希望您加入，因为对 Shrooms 的热爱，因为它在召唤您。我们长期来这里是为了一起体验这种体验。
