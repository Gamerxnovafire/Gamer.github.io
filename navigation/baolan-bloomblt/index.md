# 宝兰-Bloom(BLT)

宝兰是一个完全基于以太坊区块链的端对端模式体系（Bloom Protocol ），用于身份验证、风险评估和信用评分。宝兰能够运用去中心化技术平台体系，改善个人信息收集和评估体系，使传统货币和数字货币的贷款方，为目前尚没有银行帐户或缺乏信用评分的数十亿人群提供有效服务。
